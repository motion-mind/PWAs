Enable/disable KNX/IP tunneling on DXR2 and PXC3

With the firmware for DXR2 and PXC3 (V01.21.194.xxx and newer), the functionality “KNX/IP tunneling” is disabled by default for network security reasons. When DXR starts, the KNX/IP protocol is disabled. For commissioning or diagnostic purposes, it is enabled temporarily in the web interface. The KNX/IP protocol is disabled again the next time the DXR2 restarts.

This functionality is in Disabled state on automatic restart of the device. Enabling the KNX/IP tunneling in the offline engineering has no impact and is in the functionality will be in Disabled state after download.

It is recommended to disable the KNXnet/IP protocol after finishing commissioning or diagnostics.
