[SupEvtNotif21] Suppress event notification

Program block, name / node subtype: [SupEvtNotif21] Suppress event notification

Library hierarchy: Universal equipment

Family: SupEvtNotif

Configurator

Program blocks are stored in the library without I/Os. Use the auto-create and auto-connect functions to create I/Os and/or to connect them to the interface blocks, e.g., R_A, CMD_B. Alternatively, take the I/Os from the folder containing the predefined I/Os in the library and/or from the plant and connect them to the interface blocks.

Function

This program block represents the standard for devices with version < 1.5. For devices with version >= 1.5, use SttUpAlmHdl22 and its subsets.

Suppresses the event notification for all objects in a structured view, e.g., an automation station.

When activated over a configuration value or optional switch, the logic writes to the property 4972 of the object AsSta which is in the "Infrastructure" of the automation station.

This program block is an alternative to SupEvtNotif22 in SysFnctAS21x.

An unsuppressed value object generates an alarm that the event notification suppression is active.

This program block has the following optional functions:

Option: Suppress event notification active lamp (1xBO)

Commands a binary output that turns on a lamp, e.g., at the front of the electrical cabinet.

Option: Suppress event notification switch (1xBI)

Reads the signal from a switch and activates the event notification suppression.
