[HCChovr21] Heating/cooling changeover function

Program block, name / node subtype: [HCChovr21] Heating/cooling changeover function

Library hierarchy: Universal equipment

Family: HCChovr

Project tree

Diagram

Configurator

Program blocks are stored in the library without I/Os. Use the auto-create and auto-connect functions to create I/Os and/or to connect them to the interface blocks, e.g., R_A, CMD_B. Alternatively, take the I/Os from the folder containing the predefined I/Os in the library and/or from the plant and connect them to the interface blocks.

Function

Determines a heating or cooling state based on different conditions and sources.

The calculated heating or cooling state is displayed with a multistate calculated value and can be read from other functions, e.g., from the heating/cooling coil in an air handling unit.

HCChovr is used in the heating/cooling circuit plant (in HCcr21x in the program block HCcrPltMod21).

This program block has the following optional functions:

Option: Heating or cooling determination according to outside air temperature

Reads the outside air temperature, filters it, compares it with the temperature set with the configuration value object "Changeover point to cooling", and determines a heating or cooling state. This is coordinated with a heating or cooling request coming from other parts of the control program.

Option: Heating/cooling request

Interface prepared for a BACnet reference to an object that provides a heating/cooling request, likely in a different automation station.

Option: Heating/cooling switch (1xMI=2xBI)

Reads the signal from a switch for manual selection of heating or cooling state.

Option: Manual heating/cooling selection

Multistate configuration value for manual selection of heating or cooling state.

Option: Provide changeover condition to RDx thermostat

Interface to write to two BACnet objects for the logic for changeover condition for heating and cooling zone.

Option: Switch to heating upon frost protection

Interface to connect the signal for frost protection from the hydraulic circuit. This is an important option if HCChovr21 is used in a heating/cooling circuit plant.
