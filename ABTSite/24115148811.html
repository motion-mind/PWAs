[CK_COV] Change of value (COV) check for two multistate and two real value

Function block CK_COV checks 2 multistate values and 2 real values for a change of value.

CK_COV checks 2 multistate values and 2 real values for a change of value. The results are pending on the output.

Functioning

CK_COV compares all input values with their value for the last cycle. For a change of value, [PlsCovInd] is set to = 1 (one cycle in length).
The input values are switched directly to the outputs.
