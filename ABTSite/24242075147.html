[CenSsnCmp24] Central seasonal compensation 24, summer/winter, setpoint shift

Program block, name / node subtype: [CenSsnCmp24] Central seasonal compensation 24, summer/winter

Library hierarchy: Coordination

Family: CenSsnCmp

Project tree

Diagram

Configurator

Program blocks are stored in the library without I/Os. Use the auto-create and auto-connect functions to create I/Os and/or to connect them to the interface blocks, e.g., R_A, CMD_B. Alternatively, take the I/Os from the folder containing the predefined I/Os in the library and/or from the plant and connect them to the interface blocks.

Function

CenSsnCmp24 calculates shifts for room setpoints for comfort based on the characteristic curve and depending on the outside air temperature.

Further information

CenSsnCmp24 evaluates the outside air temperature and calculates the Comfort temperature setpoints shift for heating (SpShftHCmfCmp) and cooling (SpShftCCmfCmp).
