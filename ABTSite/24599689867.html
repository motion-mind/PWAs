Creating a notification class report

  1. Go to Building > Building structure.
  2. Open the Application & device overview task.
  3. Select Device types > Controllers > [device] or a higher building level.
  4. Right-click the device and select Reports.
  5. The Select report dialog opens.
  6. Select the Notification class report.
    Types of reports
  7. Click OK.
  1. The report is performed and visible in the report area.
