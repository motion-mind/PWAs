Use case 1: No offline ABT Site engineering data available on the PC/laptop (Work online)

Changes made online in the engineering data cannot be saved back to the project database.

 

1. Create an online project (Work online)

  1. The engineering name is entered in Settings.
    Note: this is not mandatory but recommended, that a user can see who loaded Engineering data to the device.
    Settings
  2. The Enable engineering data on device check box is activated.
    Common settings tab
  3. The Projects task is selected.
  1. Click Work online.
  2. Do the following:
    • Enter a name.
    • Enter a description.
    • Enter an administrator password.
      Note: The password must correspond with the original project password. If a password other than the original project password is used, a user and password query is displayed when the engineering data are uploaded.
  3. Click OK.
  1. An empty Work online project is created and the Startup tap opens.

 

2. Connect the project

  1. The Enable engineering data on device check box is activated.
    Common settings tab
  2. To upload the engineering data, a full download must be available from the last time.
  1. Go to Startup.
  2. Open the Configure and download task.
  3. Click Connect .
  4. The connection status changes to / .
  5. The devices are covered and displayed in the Discovered devices tab.
  6. In the Engineering data tab, the status of the engineering data is displayed.

Shows the engineering data status of the device:

  • Device does not support engineering data on device
  • No engineering data after full download available
  • Engineering data are outdated after play/pause function is used (upload of engineering data still possible)
  • Engineering data are valid after full load

 

3. Uploading engineering data

  1. Select the device you want to upload. Use the Name filter to reduce the table view.
    Note: The first uploaded device defines the building hierarchy, therefore multi-select is not available for the first device. After successful upload of the first device, other devices can be uploaded in one job (multi-select).
  2. Right-click the device and select Upload engineering data.
    • (Optional) If a password other than the original project password is used, a user and password queries displayed when the engineering data are uploaded. Enter the user name and password of the original project.
  3. Engineering data are uploaded and the status is displayed in the Engineering data tab.
    Note: Uploaded data overwrite existing engineering data. There is no merge between data on ABT Site and from the device!
  4. The building structure is created in the Engineered devices tab for the first time.
    Note: If another device with a different project structure is uploaded, a mismatch error message is displayed.
  5. Current activities are displayed in the Activities tab.
  6. Current status of the engineering data.

If a device that has been engineered with an older version of ABT Site is opened in the Programming editor, data of that device will automatically be converted.

 

4. Uploading parameter data

  • In the Engineered devices tab, right-click the device and select Upload.
  1. The current parameters of the device are uploaded.

Dynamic objects that have, for example, been created in Desigo CC, are imported and saved in a separate project folder. The dynamic objects are not visible in ABT Site and can therefore not be edited. Those dynamic objects will be downloaded the next time a download is performed.

 

 

5. Modify the engineering data in the Programming editor

  1. Right-click the device and select Go to programming or Go to engineering.
  2. Modify the control program according to the project request.

When you close the editor after a program modification, you will be prompted to load the engineering data to the device. If the user selects Cancel, he will then be responsible to download engineering data on user demand.

If one or more devices of the type PXC4.E/M (less memory) are selected, a prompt appears before the download starts. This query does not apply to the PXC4E/M-2 (more memory) devices.

 

6. Engineering/commissioning

  1. Click Play or perform a full download.
  2. A confirmation dialog displays.
  3. Click OK.
  4. The changed program is downloaded.
  5. Verify the changed program.

 

When using the play/pause function, the status of the engineering data on the device is set to Outdated engineering data. It is therefore mandatory to download the engineering data.

 

7. Download the engineering data

  1. A full download was performed successfully.
  2. The Engineering editor must be closed otherwise an error message is displayed.
  1. Go to Startup.
    Configure and download
  2. Open the Configure and download task.
  3. In the Engineered devices tab, select the updated device.
  4. Click Load > Engineering data.
  1. The engineering data are downloaded.
  2. The engineering data status changed in the Discovered devices tab from to .
  3. The Engineering data tab shows that the engineering data are downloaded and valid.
