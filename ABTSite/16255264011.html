Importing Pack & Go files

In order to change a Pack & Go project you need to import and open the project in ABT Site.

 

Import a Pack & Go project

  1. You have created a Pack & Go file.
    Creating a Pack & Go file
  1. Open Project Manager > Projects.
  2. Click Import.
  3. Select Pack & Go.
  4. Select the Pack & Go file from the default export location.
  5. Click Open.
  1. The Pack & Go package is imported.
  2. You can find the project in Project Manager > Projects, in the Pack & Go tab.
