Human-Centric Lighting topology

Human-Centric Lighting (HCL) topology

Key

1, 2

The schedulers for color temperature, brightness adjustment and associated ramping functionality for smooth transitions are located in a Desigo PX automation station.

3

The Desigo PX feeds the HCL central function (from CenFnct11) to the DXR2/PXC3 central function automation station via a BACnet reference.

4

The HCL central function in the DXR2/PXC3 communicates with the DRA room lighting application on PXC3 automation stations via grouping.

5

The adapted room lighting applications support HCL via a DALI device type 8.
