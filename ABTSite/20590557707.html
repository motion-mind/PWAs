Creating an alarm

Creating an alarm is only possible in application functions but not in the I/O device view.

Create an alarm

  1. In the Room segment editor, open the application function you want to create an alarm for.
  2. Select the I/O device.
  3. Tick the Alarm checkbox to activate the alarm for the selected I/O device.
  4. An alarm log object is created automatically.
  5. The default Object Properties for the object are set and listed in General > Compact view. Default properties can be changed as required.
