Changing BACnet object properties (CSV)

Create data export

  1. Go to Building > Building structure.
  2. Select a device or a hierarchy object.
  3. Right-click and select Create export.
  4. The Create export dialog box opens.
  5. Select the type of content to export. Click BACnet object properties (PXC4/5/7 only).
    - (Optional) Enter a comment.
  7. Click Ok.
  8. A Confirm export dialog box opens, requesting the user to confirm the action.
  9. Click Yes.
  10. The selected content is exported to the default directory for exports [drive]:\Users\Public\Documents\Siemens\DESIGO\Exports.
  11. The Create export dialog box displays allowing you to open the folder with csv file.
  12. Click Open folder.
  13. The file explorer opens.
  14. Extract the created zip.file.
  15. For each device is a csv.file created with the corresponding device name.
