Changing calculation method of room temperature sensors

You can either duplicate a room temperature sensor and modify its calculation method in the function block or use the existing configuration in the room coordination application function.

Change the calculation method of a room temperature sensor in room coordination function

  1. In Project navigation view, click room folder %R% and open the Room HVAC coordination application function in Room editor.
  2. On the Properties tab, navigate to Configuration section.
  3. You can change room temperature, room humidity and room air quality evaluation settings.

Change the calculation method of a room temperature sensor in the chart

  1. Right-click the Room temperature I/O and select Show in program.
  2. The Programming editor opens.
  3. In the Programming editor, the function block where the room temperature is calculated is in focus.
  4. To change the calculation method, select a function from the drop-down menu.
  1. Calculation method of the room temperature sensor is changed.
