Add a M-bus network

You can add an M-bus network to commission an M-bus device.

  1. Go to Engineering > M-bus.
  2. In the M-bus area, select Networks.
  3. You can add a network via two mechanisms:
    • Right-click the network and select Add network. Info: Depending on the used device, up to 5 networks for Modbus, M-bus, KNX PL-Link are supported.
    • Click Add network button above the M-bus tree.
  4. The Add network dialog box displays.
  5. Select the M-bus port in the Port field.
  6. Edit the description in the Description field.
  7. Edit the name in the Name field.
  8. Enter an extension to the M-bus name in the Extension field.
  9. The extension is added in parenthesis in the Name and field.
  10. Select the maximum rate of baud from the drop-down list in the Baud rate field.
  11. Click OK.
  1. You have added a M-bus network.
