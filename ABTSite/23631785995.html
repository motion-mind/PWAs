BACnet/IP disabled – BACnet/IP routing enabled - Hub

Configuration ABT Site – BACnet/SC

Shows the configuration in ABT Site. The floor hub AS_21 enabled BACnet/IP but diabled BACnet/IP. Although BACnet/IP routing is enabled, the subordinate node AS_22 is not visible on BACnet/IP.

 

Topology and data communication

Illustrates which devices can communicate data as well how the data is transmitted. Enabling BACnet/IP on device AS_32 – XWP hub render the device with BACnet objects visible via BACnet/IP. Although BACnet/IP routing is activated on floor hub 2nd floor, neither the hub, nor the floor node on the 2nd floor are visible on BACnet/IP. BACnet/IP must be enabled on the device for BACnet/IP routing to work.
