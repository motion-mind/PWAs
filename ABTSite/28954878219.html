Configuring a RDG2 window contact or/and presence detector

In order for the window contact or presence detector to be displayed on the RDG2 device, the corresponding parameters must be set accordingly. In a further step, the control program must be connected to the window contact and the presence detector contact.

Configure a window contact

  1. A RDG2 KNX PL-Link device is added.
  1. In the KNX PL-Link area, select the device.
  2. Select the Configuration tab on the right.
  3. The Configuration view shows the properties of the selected device feature.
  4. In the Name column, select Inputs > P153: Input 2.
  5. In the drop-down list, select Window contact.
  6. In the Name column, select Inputs > P154: Normal position and sensor, input 2.
  7. In the drop-down list, select Open or Closed.
  8. As soon as the window contact is activated, ABT Site creates a corresponding window contact BACnet object in the Programming editor.

Configure a presence detector contact

  1. A RDG2 KNX PL-Link device is added.
  1. In the KNX PL-Link area, select the device.
  2. Select the Configuration tab on the right.
  3. The Configuration view shows the properties of the selected device feature.
  4. In the Name column, select Inputs > P155: Input U1.
  5. In the drop-down list, select Presence detector.
  6. In the Name column, select Inputs > P156: Normal position and sensor, input U1.
  7. In the drop-down list, select Open or Closed.
  8. As soon as the window contact is activated, ABT Site creates a corresponding presence detector contact BACnet object in the Programming editor.
  9. Adapt the device chart for the window contact or/and presence detector.
