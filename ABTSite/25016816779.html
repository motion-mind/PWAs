Manually configuring the device on premise

Depending on the device used, the dialogs may be displayed differently. For mobile devices, it is recommended to work in landscape mode.

Connecting to the device

  1. The ABT Site project password is known and fulfill the password policy of:
    - min. 8 characters
    - upper and lower case letters
    - digits
    - special characters
    User profiles > Common settings
  1. Push the SVC button on the PXC for 2 seconds to activate the WLAN.
    Functionality of the PXC device
  2. The blue LED is blinking.
  3. The device appears in the OTHER NETWORKS section.
  4. Select the device network to establish the connection to the device.
  5. Search and connect to the PXC’s WLAN network with your PC/tablet/smartphone (SSID will be the “hardware name_xxx”, for example, PXC5.E003_0002294).
  6. Password to network is the serial number printed on the PXC (S/N: xxxxxxx).
  7. The blue LED lights up constantly.
  8. Open a web browser and enter 192.168.252.1
  9. Non-secure connection information is displayed.
  10. By confirming the information display, the password dialog opens.

 

The LED states belongs to the device types:

  • PXC4.Exxx, PXC5.Exxx, PXC7.Exxx
  • PXG3.W100-2, PXG3.W200-2

Change the initial device password

  1. The ABT Site project password is known and fulfill the password policy of:
    - min. 8 characters
    - upper and lower case letters
    - digits
    - special characters
    User profiles > Common settings
  1. In the User name field, enter Administrator.
  2. In the Password field, enter the default password OneBT used in all devices.
  3. Select Log in.
  4. The password section opens
  5. In the Enter your current password field, enter the default password OneBT used in all devices.
  6. In the New password field, enter the ABT Site project password.
  7. In the Confirm password field, enter the ABT Site project password again.
  8. Select Change password.
  1. The device password is changed to the ABT Site project password.

 

Entering the device settings

  1. The Device dialog is open.
  1. In the Device name field, enter the device name.
  2. Copy the information to the Excel table for the data exchange with Siemens.
  3. In the Device instance field, enter the device instance ID.
  4. Copy the information to the Excel table for the data exchange with Siemens.
  5. In the Description field, enter the device description.
  6. In the Location field, enter the location of the project.
  7. From the Serial number field, copy the information to the Excel table for the data exchange with Siemens.
  8. Select Next.

 

Entering the network settings

  1. The Network dialog is open.
  1. In the Configure IPv4 drop-down list, select Manual.
  2. In the Device name on IP drop-down list, select the device name.
  3. In the IPv4 field, enter the IP address for this device.
  4. Copy the information to the Excel table for the data exchange with Siemens.
  5. Select Finish.
  6. Select .
  1. The configuration is finished.

 

Activating the Cloud settings

  1. Select the Object view tab.
  2. Select Device > Infrastructure.
  3. Select Network port for Cloud.
  4. In the Enable connectivity drop-down list, select Yes.
  5. Select .
  6. In the Enable BACnet tunnel drop-down list, select Yes.
  7. Select .
  8. From the Activation key field, copy the information to the Excel table for the data exchange with Siemens.
  9. Select Activate.
  1. In the Network port for Cloud row, the status changes to Connected.
