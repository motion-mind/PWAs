Working with the user library

You can create additional user defined custom libraries with content from your local computer.

Add a user library

  1. Click the icon (three dots) in the upper right-hand corner of the Library and select Add custom library.
  2. Select the folder with the library content on your computer.
  1. A folder with your user library is created in the Library.

Remove a user library

  • Right-click the user library and select Remove.

Refresh a user library

  • Right-click the user library and select Refresh.

Open properties of a user library

  • Select the user library in the Library.
  1. The Preview pane shows the properties of the user library.
