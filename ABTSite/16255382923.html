Importing Pack & Return files

In order to merge a Pack & Return project with the original project you need to open the project in ABT Site.

 

Import a Pack & Return project

  1. You have created a Pack & Return file.
  2. The original projects exists.
  3. All the devices exist in the project.
  4. The included devices are locked in the original project.
  5. The locked devices are locked by the same Pack & Go.
  1. Open Project Manager > Projects.
  2. Click Import.
  3. Select Pack & Return.
  1. The project is imported.
  2. The devices are unlocked.
