Creating an event enrollment object - Plants navigation view

There are three options available in the Plants navigation view to add an event enrollment object.

  1. Go to Engineering > BACnet references.
  2. There are three options available in the Plants navigation view to add a trend log object. Go to the Plants navigation view:
    • Click Add data point and from the drop down select Add event enrollment.
      OR
    • Select the Plant, right-click and select Add data point > Add event enrollment.
      OR
    • Select the Folder under the Plant, right-click and select Add data point > Add event enrollment.
  3. The Add event enrollment dialog box displays.
  4. Enter the details in the required fields and click OK.
  2. The event enrollment object is created in the Plants navigation view.

The following properties can be edited:

  • Description
  • Name
  • Reference device instance
  • Reference object type
  • Reference object instance
