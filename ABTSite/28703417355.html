Workflow when RDG2 devices must be created first

For the graphical representation of a RDG2 device in Desigo CC, the data points for the Logical View must be correctly assigned to the RDG2 devices.

1. Create the first room and a RDG2 device

  1. The RDG2 devices are not created.
  2. Open the KNX PL-Link task.
  1. In the Plants navigation view, right-click and select Add plant.
  2. In the Add plant dialog box, enter a name and a description, for example:
    • Room 101
    • R_101
    • Click OK.
  3. Select the Library view on the right.
  4. From the Field devices > KNX PL-Link folder, drag a RDG2 device from the library to the new created plant (room).
    Adding and renaming RDG2 manager device
  5. Right-click the device and select Rename.
  6. Enter the name for the device, for example, Room 101.
    • Click OK.
  7. The RDG2 device is created and assigned to the selected Room folder.
  8. Select the Configuration tab on the right and configure the RDG2 device.
    Configuring the RDG2 application type
    Configuring the RDG2 heating/cooling coil demand with zones

2. Move the data point to the plant (room)

  1. In the Plants navigation view, select all the data points below the Room .
  2. Drag all the data points to the Room folder.
  3. The data points are moved below the plant. In a room with only one device, the symbol changes to white. If there are subordinate devices, the symbol remains black.

3. Create the rooms 2 - n

  1. In the Plants navigation view, select Plant.
  2. Right-click and select Add plant.
  3. In the Add plant dialog box, enter a name and a description.

4. Duplicate a RDG2 device

  1. The source RDG2 device is configured.
  1. In the KNX PL-Link area, select the RDG2 device.
  2. Right-click the device and select Duplicate.
  3. All settings and the associated control program are duplicated.
  4. Right-click the device and select Rename.
  5. Enter the name for the device, for example, Room 102.
    • Click OK.
  6. Drag the room to the desired plant .
  7. Move the data points described in the workflow step 2 (Move the data point to the plant (room)).

5. Assign the application type to the folder

The application can already be assigned when creating a plant .

  1. In the Plants navigation view, select the Room folder .
  2. Select the Properties tab on the right.
  3. The Properties view shows the plant properties.
  4. Select the Node subtype property.
  5. Enter the application abbreviation, for example, RDG_RCG_2PP_H.
    Desigo CC graphics mapping for RDG2 devices
  6. The room is configured for the Logical View in Desigo CC.

6. Setup the building structure for the rooms

  1. Go to Building > Building structure.
  2. Select Show rooms & plants .
  3. The rooms are displayed in a flat hierarchy.
  4. Drag the rooms to the correct building hierarchy.

    Note: The plant symbol is also displayed for a room. The room symbol is not supported in this version.
