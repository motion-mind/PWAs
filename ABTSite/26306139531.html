[ChHpu25] Chiller, with heat recovery/heat pump operation, with external refrigeration circuit control

Program block, name / node subtype: [ChHpu25] Chiller, with heat recovery/heat pump operation, with external refrigeration circuit control

Library hierarchy: Cooling and refrigeration equipment

Family: Hpu

Project tree

Diagram

Configurator

Program blocks are stored in the library without I/Os. Use the auto-create and auto-connect functions to create I/Os and/or to connect them to the interface blocks, e.g., R_A, CMD_B. Alternatively, take the I/Os from the folder containing the predefined I/Os in the library and/or from the plant and connect them to the interface blocks.

Function

Controls a chiller that is optimized for heat recovery/heat pump operation.

There are settings in the program for the nominal power of the chiller and for the transient state times (signal that the chiller is in a transient state that blocks the logic for generating further step-up or step-down pulses for the power control and compensation) when switching on or off. For example, 5 minutes when switching on and 2 minutes when switching off.

An interface to the control unit of the chiller (HpuItf25) enables it, sends a setpoint (it changes following the signal for heating or cooling priority) and reads optional signals for fault, feedback, defrost status, and present power.

The switch-on and switch-off sequence for the condenser, the evaporator, and the interface to the control unit are defined in the function block CMDSEQ_B.

This program block has the following optional functions:

Option: Condenser (4xAI, 6xBI, 1xAO, 3xBO)

Controls a condenser that is optimized for heat recovery/heat pump operation (Cds22).

In case of demand for hot water, it controls the outlet temperature. Without demand for hot water, it controls the inlet temperature and sends a signal that recooling is needed. There is also an optional pressure control.

Option: Evaporator (3xAI, 7xBI, 1xAO, 2xBO)

Controls an evaporator that is optimized for heat pump operation (Evp22).

It controls the outlet temperature if there is a demand for chilled water.

Option: Operating mode switch (1xMI=2xBI)

Reads the state of an operating mode switch, normally at the front of the electrical panel, and has the states "Auto", "Off", and "On".
