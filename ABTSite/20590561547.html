Configuring a room operator unit

You can configure room operator units according to your project requirements.

Configure room operator units

  1. You have added a room operator unit to your project.
  1. Go to the Room programming editor.
    Duplicating and programming
  2. Open the Room segment editor and select your room operator unit device.
  3. Expand the tree to see all the data points of the room operating unit device.
  4. Select the peripheral device object (PrphDev).
  5. In the Properties area, select the Configuration tab.
  6. The configuration settings of the device display.
  7. Make your changes according to your project requirements.
  8. Right-click the room operator unit device and select Show in device configuration.
  9. The Device view opens.
  10. In Properties > Device information > Customized view selection, click Layout.
  11. The Template view opens.
  12. You can directly enable or disable settings in the hardware layout of the device. Or open Properties > Display configuration to make your changes in the Display configuration.

Ensure that the property settings of the peripheral device match the properties in the device configuration.
