Working with the project navigation view

The Project navigation view contains one or more rooms and their components on one automation station.

Add an element from the Project navigation view to the Room segment editor
  • Drag an element from the Project navigation view to the Room segment editor.
  1. The element appears in the Room segment editor.
    Room segment
Add a device from a library to the project navigation view

In the Project navigation view, you can add network devices, applications and BACnet objects.

  1. In Global libraries, navigate to the device you want to add to the Project navigation view.
  2. Drag the selected device into the Project navigation view.
Delete an element
  • Right-click an element in the Project navigation view and select Delete from the context menu.
  1. The element is deleted from the Project navigation view. The element is not deleted in the Room segment editor.
Duplicate an element
  • Right-click an element in the Project navigation view and select Duplicate.
  1. The element is duplicated in the Project navigation view.
Navigate to an element in Programming editor
  • Right-click an element in the Project navigation view and select Show in program.
  1. The element is selected in the Programming editor.
    Programming editor
Open an element in the Programming editor
  • In the Project navigation view, do one of the following:
    • Double-click an element, e.g., a room.
    • Right-click an element and select Open.
    • Select an element and press the [Enter] key.
  1. The element opens in the Programming editor.
    Programming editor
Open the device configuration
  • Right-click an element in the TX & Onboard I/Os folder in the Project navigation view and select Show in device configuration.
  1. The Device configuration view opens.
    Device configuration view
Show the properties of an element
  • Right-click an element in the Project navigation view and select Properties.
  1. The Element properties in the Properties tab opens.
