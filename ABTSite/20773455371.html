Opening the Programming editor

The Programming editor is part of the Room programming editor.
Duplicating and programming

You can open application functions and data points in the Programming editor. You can also directly select a function block and navigate to the right spot in the Programming editor.

Open a function block in the Programming editor

  1. Open the Room segment editor to display the room segment table.
  2. In the application function, select the function block you want to display in the Programming editor.
  3. Right-click the function block and select Open in program.

Open an application function in the Programming editor

  1. Open the tree of a room or room segment to display the available application functions.
  2. Double-click the application function.

Open a data point in the Programming editor

  1. Select a data point in your Resources folder.
  2. Right-click the selected data point and select Open in program.
