Project navigation view

The Project navigation view contains one or more rooms and their components on one automation station.
Working with the project navigation view

Command

Description

<Name of the automation station (AS)>

Shows the AS and its components.

The icons indicate if a chart is mapped to BACnet and if a chart is locked.

Device configuration

Shows the device configuration of the controller including:

  • Controller with hardware attached
  • KNX PL-Link devices

KNX PL-Link devices are configured in this view.

BACnet objects

Shows all BACnet objects added to the controller.
BACnet objects

Rooms

Shows all rooms added to the AS.
Room

Room segments

Shows all room segments added to the controller including:

  • Data points
  • Devices
  • Applications

Room segment

In the room segment editor, you can go online with devices. You can also see all I/O devices and application functions attached to the controller.

TX & Onboard I/Os

Shows all TX and onboard I/Os added in the I/O configuration task.
Clicking on I/O will navigate to I/O in the Field programming editor.

KNX PL-Link

Shows all KNX PL-Link devices added to the rooms.
Clicking on I/O will navigate to I/O in the Field programming editor.
