System of units

Function blocks depend on a system of units for calculations. They display it on the output System of units [SysUnits].

Function blocks rely on the device object property "System of units" to adapt internal calculations to the system of units in use.

If the device object property "System of units" is unavailable or cannot be accessed, or its value is not recognized, the function block uses the "International system of units (SI)" and indicates this with the value "1: Internat. system of units (SI) - fallback" on the output [SysUnits]. In these cases, the outputs [ErrCode] and [RErrNr] supply additional information.

The following systems of units are available:

  • International system of units (SI) (fallback)
  • International system of units (SI)
  • US system of units
  • Imperial system of units
  • Canadian system of units
