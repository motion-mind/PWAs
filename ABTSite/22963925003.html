[MdltPuGrp21] Modulation for pump group

Program block, name / node subtype: [MdltPuGrp21] Modulation for pump group

Library hierarchy: Universal functions

Family: MdltPuGrp

Project tree

Diagram

Configurator

Program blocks are stored in the library without I/Os. Use the auto-create and auto-connect functions to create I/Os and/or to connect them to the interface blocks, e.g., R_A, CMD_B. Alternatively, take the I/Os from the folder containing the predefined I/Os in the library and/or from the plant and connect them to the interface blocks.

Function

Provides logic to be used in a pump group with modulating pumps that determines how many pumps are switched on and what their speed is based on the number of available pumps and 0-100 % control signal.

There are settings for the minimum and maximum speeds for the relevant pumps which work in parallel.

Use the minimum and maximum speed settings from the data sheet (pressure for given volume flow at a given speed) and adjust them if the pressure setpoint changes.

A parameter in the program defines the maximum speed of the last pump in the group. Its default setting is 100 %.

This program block has the following optional functions:

Option: 2 pumps...8 pumps

Check how many pumps are in the pump group. Leave all the options with lower and equal number of pumps active. Delete the options with higher number of pumps. For example, in case of 4 pumps, leave the options "2 pumps", "3 pumps" and "4 pumps".
