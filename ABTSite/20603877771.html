Device configuration view

The device configuration contains a graphical view of the hardware layout. And you can select and drag devices from the catalog into the Device view. The device configuration is used for setting up the assigned hardware and configuring KNX devices.

(2) Properties area

The Properties area contains detailled information on the selected item. The subsections change depending on what is selected.

Subsections when an automation station is selected:

  • General
    • Catalog information
    • Application type information
    • Unit conversion
    • TX device types
  • BACnet/IP interface
  • BACnet MS/TP interface

Subsections when a room operator unit is selected:

  • General
    • Catalog information
  • Device information
  • Functions
  • Alarm

For more information on the selected automation stations and room operator unit, see the corresponding hardware documentation.

(3) Device view / Template view

A layout view of the device. When you open a room operator unit device, the view changes, and the Template view opens.

(4) Hardware catalog and Libraries

There are two library sections.

  1. The Project library contains items specifically used in the current project.
  2. The Global libraries are part of the application template and can be used in all projects.
