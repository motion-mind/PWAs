③ Toolbar

Command

Description

State

Indicates the state of the application. The green color indicates that the application is online and running or paused.

  • Gray: Engineering session not active, Not connected
  • Green: Application running
  • Orange: Download required
  • Red: Download in progress, Backup in progress or Busy

Check

Checks for duplicate names, number of I/Os, trend limits, etc. The check is automatically performed before loading a device.
Showing data point limits when compiling

(Play)

Only visible when the device is online. Activates an online session and loads changes for the selected device.
Using play/pause function

(Pause)

Only visible when the device is online and an online, session is started. Deactivates an active online session.

 (Alarming)

Temporarily suppresses all alarms for testing purposes.
Suppressing alarms during commissioning or maintenance

Note: If several alarms are triggered due to a field device status is “Device-Missing” (perform an alarm shower), only the “Device-Missing” alarm is sent to the corresponding recipients. This ensures that the user does not receive and have to acknowledge multiple alarms.

④ KNX PL-Link area

In the KNX PL-Link area, there is a network tree for navigating the elements of the KNX PL-Link network. There is also separate toolbar, and the workplace table. To add new KNX PL-Link devices, drag them from the library.
Adding a QMX devices

Command

Description

Expand/collapse all KNX PL-Link elements.

Estimated power consumption bar. Shows a bar of the actual power consumption compared to the maximum possible consumption.

Add network

Opens the Add network window for creating a new network.

Add device

Creates a device below the network. One or more subordinate devices can be added as well.

The Workplace table provides an overview of the KNX PL-Link network. Click an item in the network tree to view its properties. There are different columns, depending on the selection.

Column

Description

Name

The name of the element. Editable text field.

Description

The description of the element. Editable text field.

Status

The status of the selected network.

Unit

The type of unit.

Type

The type of object.

 (Alarming)

Select to activate the alarming function for the object.
In the filter field, select True to filter for objects with activated alarming function.

 (Trending)

Select to activate the trending function for the object.
In the filter field, select True to filter for objects with activated trending function.

⑤ Library, Properties and Configuration

The Library and Properties area on the right can be toggled. The Library view contains elements which can be dragged directly to the working area.
Library

The Property view shows the properties of the selected element. The items displayed in the Property view depend on the selected object type.

The Configuration displays a layout of the device when selected.
