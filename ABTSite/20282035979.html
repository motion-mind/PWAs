[AQualCtr21] Air quality controller, control of the mixed air damper and 2nd fan speed

Program block, name / node subtype: [AQualCtr21] Air quality controller, control of the mixed air damper and 2nd fan speed

Library hierarchy: Ventilation and air conditioning functions

Family: AQualCtr

Diagram

Configurator

Program blocks are stored in the library without I/Os. Use the auto-create and auto-connect functions to create I/Os and/or to connect them to the interface blocks, e.g., R_A, CMD_B. Alternatively, take the I/Os from the folder containing the predefined I/Os in the library and/or from the plant and connect them to the interface blocks.

Function

Performs air quality control.

The minimum controller output is influenced by the setpoint for minimum outside air (from the mixing dampers).

This program block has the following optional functions:

Option: Demand for fan full speed

If the output of the controller goes over a specific value (e.g., 80 %) demand for fan full speed is generated and this is normally used to switch the fans to speed 2.
