Working in the graphical rail view

  1. Go to Building > Building Structure.
  2. Right-click the automation station and select Go to engineering.
  3. Open the I/O configuration task.

Zoom In/out

  1. Go to the I/O configuration area.
  2. To collapse graphical view of the rails, click and to expand the graphical view of the rails, click .
  1. You have expanded/collapsed the view of the rails.

Graphical/Tabular view of rails

  1. Go to the I/O configuration area.
  2. Click the Tabular view tab to see the tabular view of the rails and click the Graphical view tab to see the graphical view of the rails.

Online rail engineering

The data points in online rail engineering appear in the following colors:

  • Data point - Selected
  • Data point - Okay
  • Data point is manual/local override
  • Data point is alarm/faults.

Move modules or accessories on rails

  1. Click the Graphical view tab on the I/O configuration area.
  2. Drag the module or accessory you want to move to another position within the same rail or another rail.
  1. You moved the module or accessory to another position.
    INFO: The position of the PXC device as well as of the bus interface modules TXA1.IBE cannot be changed.

Move rails up/down

  1. Select the rail that you want to move up or down.
  2. To move the rails up and down, click for moving the rail up and click for moving the rail down.
  1. You have moved the rails up or down.

Change connection to other rails

Local rails can be connected to any rail. Decentralized rail can be connected to main or other decentralized rails.

  1. Select Connected to of the rail that you want to change the connection to another rails.
  2. Select the rail you want to connect to in the window.
  1. You have changed the connection to other rails.

Delete rails

  1. Click the Graphical view tab on the I/O configuration area.
  2. Select the rail that you want to delete.
  3. Click on top left of the rail.
  1. You have deleted the rail. Any belonging modules or accessories are moved to other rails. Automatically created bus interface modules TXA1.IBE are deleted. The main rail cannot be deleted.

Show assigned data point in module or onboard I/O

  1. Go to the Plants navigation view.
  2. Select the data point that you want to view on the module.
  3. Right-click the selected data point and select Show in module.
  1. The data point is highlighted on the module.

Add a new data point from the library to an I/O module

  1. Click the Graphical view tab on the I/O configuration area.
  2. In the Plants navigation view, click to set the drop destination focus new.
  3. The data points will be assigned to this plant .
    Note: It is always possible to drag a data point directly to the position in the Plants navigation view, where its functionally belongs too (for example, valve).
  4. Select the Library tab.
  5. Select the data point you want to add to an I/O module.
  6. Drop the data point into a slot that fits. Free or compatible I/O slots are displayed in green.
  1. You added a new data point to an I/O module.

Move a data point

  1. Click the Graphical view tab on the I/O configuration area.
  2. Select the data point you want to move to another hardware slots within the same module or of another module or onboard I/O.
  3. Hover the point over empty hardware slots within the same module or of another module. Empty slots that fit for the data point are indicated in green. Not possible slots are shown in red (see video on top of this section).
  4. Drop the point into a slot that fits.
  1. You moved the data point to another hardware slot.

Unassign data points from the module onboard I/O

  1. Click the Graphical view tab on the I/O configuration area.
  2. Select the assigned data point that you want to unassign.
  3. Click × on the data point.
  1. You have removed the data point on the module onboard I/O.

Delete a data point

  1. Click the Graphical view tab on the I/O configuration area.
  2. Select the data point that you want to delete in the Plants navigation view.
  3. Right-click the selected data point and click Delete.
  1. You have deleted the data point.

Delete modules or accessories from rails

  1. Click the Graphical view tab on the I/O configuration area.
  2. Select the module that you want to delete from the rail.
  3. Click .
  1. You have deleted the module or accessory from the rail. Any assigned data points of the deleted module will be unassigned.

You have to delete the belonging rail in order to delete the automatically created bus interface modules TXA1.IBE.
