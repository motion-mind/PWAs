Configuring the presence detector manager

  1. A presence detector manager device is available.
  1. In the KNX PL-Link area, select the device.
  2. Select the Properties tab on the right.
  3. Modify the properties as required.
  4. Select the Configuration tab on the right.
  5. The Configuration view shows the properties of the selected device feature.
  6. Modify the properties as required.
