Using play/pause function

The play/pause function can be used for performing delta downloads.

It is recommended to perform a full download before using the play/pause function. A prior full download is especially recommended when a Modbus network is created or a BACnet/SC certificate must be renewed.

Properties for device configuration are not loaded via play/pause function.
Downloading device configurations

Use play/pause function

  1. The device is configured and assigned.
  1. Click .
  2. Online parameter changes are uploaded.
  3. The changed hardware is downloaded.
  4. The engineering editor is online.
  5. An orange frame around the I/O configuration area becomes visible.
  6. Live status and online Values/Unit of all data points are displayed.

  7. You cannot add new data points in the online session.

  8. Check and fix wrongly displayed values in data points.
  9. The Live status is updated as soon as the values are correct.
  10. Change the Commissioning state of the data points tested to passed.
  11. Click  when all changes are made.
  12. The engineering editor is offline.
  13. The changes done in play mode are written in the device and in the offline data.
  14. Verify that the changes you made are reflected offline.
  15. Click  again.
  16. Commissioning status entered in the last session is saved and visible.

When using the play/pause function, the status of the engineering data on the device is set to Outdated engineering data. It is therefore mandatory to download the engineering data.
