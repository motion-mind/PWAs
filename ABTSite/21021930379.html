Moving device instance to new custom app type

You can move commissioned DXR2 devices into updated room template without re-commissioning (including device assignment, data point test data and setpoint/parameter settings).

Devices can be moved within templates of the same device type (HW) and of the same application type.

Move device instance to new custom app type
