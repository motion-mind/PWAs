Opening the Web interface

To check the device assignment, open the Web interface.

Open the Web interface

  1. The configuration is downloaded to the device.
    Configuring and downloading to a device
  1. Open the Room programming editor.
    Duplicating and programming
  2. In the Project navigation view, right-click the device and select Web interface.
  3. The Web interface of the device opens in the same window.
  4. Click Installation.
  1. The Device overview of the Web interface opens.

For more information on the Web interface see the User guide.
