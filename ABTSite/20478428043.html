[Bu24] Burner interface, enable and modulating outputs

Program block, name / node subtype: [Bu24] Burner interface, enable and modulating outputs

Library hierarchy: Heating equipment

Family: Bu

Project tree

Diagram

Configurator

Program blocks are stored in the library without I/Os. Use the auto-create and auto-connect functions to create I/Os and/or to connect them to the interface blocks, e.g., R_A, CMD_B. Alternatively, take the I/Os from the folder containing the predefined I/Os in the library and/or from the plant and connect them to the interface blocks.

Function

Controls an analog output for modulation (modulating) and a binary enable output (command) that are sent to the boiler control unit.

This program block can be used in a program block for a boiler.

This program block has the following optional functions:

Option: Fault (1xBI)

Reads the fault signal from the boiler control unit and generates an alarm.

Option: Feedback (1xBI)

Reads the feedback signal from the boiler control unit.
