Configurator

Program blocks are stored in the library without I/Os. Use the auto-create and auto-connect functions to create I/Os and/or to connect them to the interface blocks, e.g., R_A, CMD_B. Alternatively, take the I/Os from the folder containing the predefined I/Os in the library and/or from the plant and connect them to the interface blocks.

Function

Generates a target power request for the logic for power control and compensation.

This function can accelerate the system response if the energy generation (with multiple generators) starts at low outside temperature. The logic for power control and compensation gets a target power to go to that is closer to the needed power for this outside temperature. Without this function, the system needs a longer time to reach this power because the logic for step-up and step-down request needs to calculate and issue several pulses (and there are likely long transition times to wait for between the pulses).
