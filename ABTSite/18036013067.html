Change custom name or custom description

  1. Select Sites, then select the site from the list.
  2. Select the device.
  3. The Information tab displays.
  4. Click Edit.
  5. The Edit dialog box opens.
  6. Change the Custom name and/or Custom description.
  7. Select Save.
  1. The edited data is displayed in the device list.
