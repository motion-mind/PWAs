M-bus modes

There are two modes in ABT Site for the user to map the data points.

Preset mode

Also known as "easy mode". In this mode the user is guided by the user interface to select various parameters, some of them will be part of the I/O address through DIB/VIB.

  • Virtual properties need to be created to represents parts of DIB/VIB and with appropriate validations. These are helpful in modelling and visualization.
  • Predefined VIF tables are made available so that the user can select the desired parameters such as measured value, unit, etc.

Note: The preset mode is a virtual mode derived from the DIB and VIB which are part of the I/O address. The indication of whether it is the preset mode is NOT stored anywhere. If the entered VIB/DIB are available in our meta of presets, then we deem it as a preset. If they are not found, then we deem it as custom.

Custom mode

Also known as "expert mode". In this mode the user can enter only DIF/E and VIF/E codes.

  • Even though the user may enter any values here, if the user switches to the preset mode, ABT Site re-calculate the values from the DIB/VIB.
  • If the recalculation fails, then the corresponding virtual properties shows as "Custom".
