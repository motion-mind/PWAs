Overwriting values online

Values can also be directly overwritten. If you change values, the Present priority column indicates that the corresponding value has been overwritten manually. You can release manually overwritten values in the Web interface.

Overwrite values online

  1. Open the Room programming editor.
    Duplicating and programming
  2. In the toolbar, select the Go online button.
  3. The frame color changes to orange, and the table in the Room segment editor now displays live values.
  4. In the Room segment editor table, select the value.
  5. Enter the new value.
  1. The Present priority column indicates that the corresponding value has been overwritten.

Overwrite values online and remove overwrite changes on the device are displayed immediately in the table. Values can also be directly overwritten.

Remove overwrite in Web interface

You can either undo the overwrite operation or release manually overwritten values in the Web interface.

For more information on removing overwrites, see Device and network configuration in the Web interface user guide.
