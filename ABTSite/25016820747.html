Loading the device over the Cloud

Connecting to the Cloud from office

  1. The Ethernet connection is disconnected.
  2. The Cloud Link is operating.
  3. The device on the customer site is connected to the Internet.
  1. Go to Startup.
  2. Open the Configure and download task.
  3. In the Engineered devices tab, select the device.
  4. Select the arrow symbol to the right of Connect .
  5. The Connection settings dialog box opens.
  6. Select the Cloud tab.
  7. Select Connect.
  8. The Cloud Log In dialog box opens.
  9. Enter your personal log-in data.
  10. Establishing a connection to the cloud takes a certain amount of time.
  11. After connecting to the Cloud, the device is visible in the Discovered devices and Devices in the Cloud tabs.

 

Loading device data from office

  1. The discovered device is visible in the Discovered devices tab.
  1. Select the device in the Engineered devices tab.
  2. Right-click and select Load > Network configuration (IP/MS/TP).
  3. Right-click and select Load > Full (device may restart).
  4. (Optional) Right-click and select Load > Engineering data.
  5. (Optional) Right-click and select Load > Building X integration data.
    Note: Those data are used for the Building X Operation manager.
  1. The device data are loaded and ready for commissioning or Building X engineering.
