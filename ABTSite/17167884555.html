Connecting via a cloud connection

You can connect to a device or network via a cloud connection. You must be a registered user of the Siemens ID service to use this feature. Siemens ID enables secure access to different Siemens applications and services. With Siemens ID, you can use one digital identity to access services provided by Siemens.

The token is valid for 24 hours from time of creation. It is therefore not possible to make trend recordings longer than 24 hours without having an interruption in the recording. More information can you find in the Building X document A6V13414788 API Manager User Guide.

 

Connect over a cloud connection

  1. The CloudLink is installed on your computer.
    For more information to operate CloudLink, please refer to the CloudLink user guide (A6V15653550).
    http://www.siemens.com/download?A6V15653550
    CloudLink as PDF
  2. You are a registered user of the Siemens ID service.
  1. In the common toolbar, select Sign in.
  2. The Siemens ID Login Service window opens.
  3. Enter your credentials and click Login.
  4. In the Common toolbar, select the arrow beside the Connect button.
    Connect and disconnect
  5. Select the Cloud tab.
  6. Click Connect.
  1. The cloud connection is established.
  2. You have access to more resources such as different application templates, which are also available in the ABT Site workflows (e.g. when adding devices from the library).
    Adding devices

For more information to onboarding the automation stations, please refer to the Desigo Engineering Tool Access documentation (A6V12096474).
