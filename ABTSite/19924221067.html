Chart objects list

The Chart objects list shows all plants and program blocks according to control program structure.

Select a plant or a program block from the Chart structure drop down-list box to display its chart objects.

Column

Description

Object type

The object type of the chart object.

Description

The description of the chart object.

Object path

The full object path of the chart object.

Show properties of a chart object
  • Select a chart object in the list.
  1. The properties of the chart object are shown in the Properties tab.
Navigate to a chart object in the Programming editor
  • Do one of the following:
    • Right-click a chart object in the list and select Show in program.
    • Double-click a chart object in the list.
  1. The corresponding element is shown in the Programming editor, if it is represented in the chart.
