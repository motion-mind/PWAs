Creating a reference in the programming editor

This task will not create a BACnet reference object. Any values of the reference are visible in the programming environment only. The task is also available for all programmable device versions lower than device version 1.3.

 

Create a reference from the BACnet reference task

  1. The Programming editor is open.
  1. Go to Engineering.
  2. Open the BACnet references task.
  3. Select an object either in task Projects or EDE import.
  4. The list of BACnet objects as source opens.
  5. You can drag and drop a BACnet object from the task into the chart of the Programming editor.
  6. You can also create the BACnet reference as follows:
    • Select a function block in the chart of the Programming editor.
    • In the Element properties tab > Block > BACnet object type select BACnet reference.
    • By using the context menu Copy a BACnet object from the task in the BACnet references and paste the content into the function block of the Programming editor.
  1. You have created a reference.

 

Create a reference in the Programming editor manually

  1. The Programming editor is open.
  1. Select a function block in the chart.
  2. In the Element properties tab > Block > BACnet object type select BACnet reference.
  3. The properties Device instance, Object type, Object instance are enabled.
  4. Enter the values manually.
  1. You have created a reference.
