Room ID follows device instance number

You can create Room /Group IDs using the BACnet device instance number.

Room devices - Room IDs

  1. Go to Settings > Number ranges and under Range settings, tick the checkbox for Room ID follows device instance number.
  1. Go to Building > Building structure.
  2. Click Add device.
  3. In the Add devices window that opens, expand the Add via application template option.
  4. Expand the Library option.
  5. Select the template for which you want to create the automation station and click Next.
  7. A new window opens to enter the device details.
  8. In the Number of new devices field, enter the number of devices you want to add.
  9. In the BACnet tab, enter the device instance number in the Start device instance number field.
  10. Enter the increment value/number in the Increment device instance number field.
  11. Click OK.
  13. The automation stations are created according to the number of devices added.
  14. Click .
  15. Select the room and open the Details pane.
  16. Click Naming.
  1. The Room ID is displayed on the right window. This Room ID follows the business logic such as, Room ID = 100 times (Device instance number + increment value).
    For example, Start device instance number = 600, Increment device instance number = 1 and Number of new devices = 3. The devices are created with the device instance numbers as 600, 601 and 602 and the Room IDs for these three devices will be created as 60001, 60101 and 60201.

Central functions - Group IDs

  1. Go to Settings > Number ranges and under Range settings, tick the checkbox for Room ID follows device instance number.
  1. Go to Building > Central functions.
  2. Click Add hierarchy element.
  4. In the window that opens, enter the Type and Count of the hierarchy element and click OK.
  5. A new hierarchy element F_01 is created.
  6. Drag and drop the central function element into the newly created hierarchy element F_01.
  7. Expand the central function element.
  1. The group number is displayed in the Group no column. This group number follows the business logic such as, Group no. = 100 times Device instance number. This is logic used for each group category.
