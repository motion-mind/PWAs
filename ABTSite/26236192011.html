Exporting the certificate signing request for the intermediate IEEE 802.1X root certificate

By using an intermediate certificate, the intermediate certificate is switched over the actual certificate and the certification authority. This means that the underlying device certificates do not necessarily have to be renewed.
Working with IEEE 802.1X certificates

  1. Go to Settings.
  2. Open the Certificates task.
  3. Select the IEEE802.1X certificates tab.
  4. Select the ABT Site root certificate, [name of the root certificate].
  5. In the Details view, click Export CSR for ABT signing certificate.
  6. The root certificate is exported and located, for example, C:\Users\Public\Documents\Siemens\DESIGO\Exports.
  7. The created certificate files can now be sent to the external trusted certificate authority.
