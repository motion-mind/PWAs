[Evp21] Evaporator, outlet temperature control

Program block, name / node subtype: [Evp21] Evaporator, outlet temperature control

Library hierarchy: Cooling and refrigeration equipment

Family: Evp

Project tree

Diagram

Configurator

Program blocks are stored in the library without I/Os. Use the auto-create and auto-connect functions to create I/Os and/or to connect them to the interface blocks, e.g., R_A, CMD_B. Alternatively, take the I/Os from the folder containing the predefined I/Os in the library and/or from the plant and connect them to the interface blocks.

Function

Controls an evaporator that is optimized for chiller operation with optional pump and mixing valve.

This program block has the following optional functions:

Option: Mixing valve (1xAO, 1xAI)

Controls the position of a mixing valve (Vlv24).

If the optional mixing valve is used, it is controlled by the following function:

Temperature control for the outlet temperature (part of the option "Mixing valve").

The outlet temperature is controlled by a PID controller that compares the sensor value with the setpoint for chilled water and changes the position of the mixing valve.

We recommend using the outlet temperature sensor.

Option: Pump (1xBO, 4xBI)

Controls a pump (Pu21). Different alternatives are possible, e.g., TwnPu21.

The pump runs when the evaporator is running, and the optional shutoff-valve is open.

Option: Shutoff valve (1xBO, 2xBI)

Controls a shutoff valve that opens when the evaporator is running (Vlv21).

Option: Flow detector (1xBI)

Reads the signal from a flow detector and generates an alarm if the evaporator is running and no flow is detected.

Option: Inlet temperature (1xAI)

Reads the signal from an inlet temperature.

Option: Outlet temperature (1xAI)

Reads the signal from an outlet temperature.
