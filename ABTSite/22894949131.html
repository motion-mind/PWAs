[OutItf21] Interface output, activate and modulating output

Program block, name / node subtype: [OutItf21] Interface output, activate and modulating output

Library hierarchy: Universal equipment

Family: OutItf

Project tree

Diagram

Configurator

Program blocks are stored in the library without I/Os. Use the auto-create and auto-connect functions to create I/Os and/or to connect them to the interface blocks, e.g., R_A, CMD_B. Alternatively, take the I/Os from the folder containing the predefined I/Os in the library and/or from the plant and connect them to the interface blocks.

Function

Controls an analog modulating output and an optional binary enable output (command).

This program block can be used in different program blocks (electric heating coil, humidifier, cooling coil with direct expansion, heating/cooling coil with direct expansion) to control their outputs, which are read by the control unit of the corresponding aggregate.

This program block has the following optional functions:

Option: Command (1xBO)

Enables the aggregate to be controlled by sending a binary signal to its control unit.

Some control units require only an analog signal, e.g., 0-10 V, and no binary output.
