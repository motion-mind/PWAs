Using the new notification class for the alarm

Create a new alarm and assign the new notification class in the Engineering menu.
Notification classes

  1. Go to Building > Building structure.
  2. Right-click a device and select Go to engineering.
  3. Open the I/O configuration task or BACnet objects task.
    I/O configuration
  4. In the Plants navigation view, select a data point.
  5. Open the Properties tab.
  6. Go to the Alarm tab.
  7. Select the Compact view option.
  8. From the Enable event drop-down list, select Yes.
  9. From the Notification class drop-down list, select the desired notification class.
  1. The alarm is assigned to a notification class and alarm-capable.
