[HGenCtl21] Sensors, setpoints and control for single heat generator

Program block, name / node subtype: [HGenCtl21] Sensors, setpoints and control for single heat generator

Library hierarchy: Heating equipment

Family: HGenCtl

Project tree

Diagram

Configurator

Program blocks are stored in the library without I/Os. Use the auto-create and auto-connect functions to create I/Os and/or to connect them to the interface blocks, e.g., R_A, CMD_B. Alternatively, take the I/Os from the folder containing the predefined I/Os in the library and/or from the plant and connect them to the interface blocks.

Function

Performs temperature control for the hot water flow temperature.

The setpoint for the hot water temperature control is increased by a small offset defined by an analog configuration value, and the slightly higher setpoint for the hot water temperature is sent to the energy generator(s).

This program block has the following optional functions:

Option: Hot water storage tank

Controls the charging state of a hot water storage tank (with 4 temperatures) (CrgReqStk21).

Option: Outside air temperature (1xAI)

Creates an analog input for outside air temperature and reads its signal.
