Changing favorite parameters online

You can change parameters of devices with preconfigured applications online.

  1. You have created a list of custom favorites.
    Creating a list of favorites
  1. Open the Startup > Configure and download task.
  2. Connect to the device.
    Connecting to devices
  3. Right-click the discovered device and select Show online favorites.
  4. Select the list of custom favorites.
  5. Select the object.
  1. The list of parameters is displayed on the left. You can change the writeable parameters online. Changed parameter values are updated directly to the online device.

You can change parameter values online per automation station in Startup > Favorites (online). But the parameter list can only be changed in Configuration > Favorites.
