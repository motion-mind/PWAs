[PuGrp26] Pump group, 1-speed, max. 8 pumps

Program block, name / node subtype: [PuGrp26] Pump group, 1-speed, max. 8 pumps

Library hierarchy: Universal equipment

Family: Pu

Project tree

Diagram

Configurator

Program blocks are stored in the library without I/Os. Use the auto-create and auto-connect functions to create I/Os and/or to connect them to the interface blocks, e.g., R_A, CMD_B. Alternatively, take the I/Os from the folder containing the predefined I/Os in the library and/or from the plant and connect them to the interface blocks.

Function

Controls a pump group of up to 8 1-speed pumps.

The input signal in % is calculated externally and must have discrete values representing stages. Each new stage activates an additional pump.

A program logic (function block ROT_8) measures the operating hours of the pumps and switches on the pump with fewer operating hours when the pump group is switched on. When the next pump is to be switched on, the logic uses these operating hours as a criterion for selecting the next pump.

If one pump has a problem, the program switches on a different pump (if available). If all pumps have a problem, a calculated value object "Pump group fault" generates an alarm.

This program block has the following optional functions:

Option: Pump 3...Pump 8

The pumps are preset to be of type Pu21. We recommend not changing them to an alternative.

Option: Rotation schedule

A short pulse that forces the net rotation can be set up, e.g., every Tuesday from 5:15 a.m. to 5:16 a.m.
