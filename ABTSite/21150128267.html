Ethernet and USB port management

For security reasons, unused Ethernet port on an automation station must be disabled. Flexible Ethernet port is not supported for DXR2 room automation stations. The current state displays in the Device details within the IP section, for example, PXC7.

Building > Building structure > Details pane > Details pane of the device version ≤ 1.5
Building > Building structure > Details pane > Details pane of the device version ≥ 1.6

 

Port management with daisy chain connection

With daisy chain wiring, care must be taken to ensure that no port is switched off that still has downstream devices. Communication is interrupted and data from subsequent devices can no longer be transmitted (red line).
