Rail properties window

Properties window

1. Modify main rail properties

  1. You want to edit the properties of the main rail.
  1. Click on the main rail.
  2. The Properties window displays to the right of the graphical view.
  4. In the Rail length field, enter the value for the length of the rail.
  5. You have adjusted the length of the rail.
  6. Enter the required name of the rail in the Name field and the description of the rail in the Description field and select the extension of the rail from the list.
  7. Enter the control panel number in the Panel field and the name of the plant in the Plant room field.
  1. You have made the required changes in the properties of the main rail.

2. Modify local rail and decentralized rail properties

For decentralized rails, an own power supply estimation displayed at the top of the decentralized rail.

  1. You want to edit the properties of local rail or decentralized rail.
  1. Click on the local rail or main rail.
  2. The Properties window displays to the right of the graphical view.
  4. In the Rail length field, enter the value for the length of the rail.
  5. You have adjusted the length of the rail.
  6. Enter the required name of the rail in the Name field, description of the rail in the Description field, and select the extension of the rail from the list.
  7. Enter the panel name in the Panel field and the name of the plant room in the Plant room field.
  8. To change the type of rail, select the type of rail in the Rail type field.
  9. To designate the connection of the rails, select the rail type in the Connected to field.
  1. You have made the required changes in the properties of the local and decentralized rail.

3. Modify module properties

  1. You want to edit the properties of the modules in the rails.
  1. Click on the selected I/O module.
  2. The Properties window displays to the right of the graphical view.
  4. Make the required changes in the Address, Enable event, Notification class, Enable event detection, Suppress event notification and Suppress event detection. For Module type, the user can only change to the possible module types that fit to the already assigned data points.
  1. You have made the required changes in the properties of the selected module.
