[CGenCtl21] Sensors, setpoints and control for single refrigeration generator

Family: CGenCtl

Configurator

Program blocks are stored in the library without I/Os. Use the auto-create and auto-connect functions to create I/Os and/or to connect them to the interface blocks, e.g., R_A, CMD_B. Alternatively, take the I/Os from the folder containing the predefined I/Os in the library and/or from the plant and connect them to the interface blocks.

Function

Performs temperature control for the chilled water flow temperature.

The setpoint for the chilled water temperature control is reduced by a small offset defined by an analog configuration value. The slightly lower setpoint for the chilled water temperature is sent to the energy generator(s).

The logic for chilled water temperature control has one fixed setpoint. The refrigeration machines and the chilled water system are usually designed for a defined temperature level and should not be influenced by consumers.

This program block has the following optional functions:

Option: Chilled water storage tank

Controls the charging state of a chilled water storage tank (with 4 temperatures) (CrgReqStk21)
