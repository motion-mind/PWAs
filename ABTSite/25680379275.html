Exporting an IEEE 802.1X operational certificate signing request

This procedure describes how to change the certification authority (CA) from internal to external for individual devices.

Export a certificate signing request together with an operational certificate for individual devices, then hand them over on file level to the external CA for signing. Import an externally signed certificate to a device in ABT Site. The CA is changed from internal to external after import..

 

Export a certificate signing request

  1. The device supports operational certificates.
  2. The device has a valid serial number.
  1. Open Building > Certificates management.
  2. Select at least one device with external certificate type.
  3. Select the IEEE 802.1X tab.
  4. Click Export CSR.
  1. The files *.csr and *.txt are exported.

    You can deliver the file to a certification authority for signing.
  2. Note: The file *.csr contains a file header. The *.txt does not have a header. Depending on your CA, one or the other may be more convenient to work with.

 

OR

 

Export a certificate signing request

  1. The device supports operational certificates.
  2. The device has a valid serial number.
  1. Open Building > Building structure.
  2. Select at least one device with external certificate type.
  3. Open the Details pane.
  4. Select Certifcates.
  5. In the X802.1 certificate section, click Export certificate signing request.
  1. The files *.csr and *.txt are exported.

    You can deliver the file to a certification authority for signing.
  2. Note: The file *.csr contains a file header. The *.txt does not have a header. Depending on your CA, one or the other may be more convenient to work with.
