Removing devices

This chapter describes the workflow for removing a device. The device gets removed from the current partition, but remains available in the system. You can add this device to a partition again and all app instance data are kept as long as the checkbox Reuse (enabled by default) is selected and the assigned distribution contains these apps. With this checkbox disabled, all application-related data are deleted.

No interaction on the device is required. Network and firewall settings are kept after assigning to a new partition.

This operation cannot be undone.

  1. You are in  Sites.
  2. The distribution is unassigned.
  1. Select the site with the device to be deleted.
  2. Select the device.
  3. In the Information tab select Remove.
  4. Confirm your decision.
