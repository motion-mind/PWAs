Handover report for port settings

Read back data
  1. The device is connected.
    Connecting to devices
  1. Go to Startup.
  2. Open the Configure and download task.
  3. Select the Discovered devices tab.
  4. Right-click the device and select Upload.
  1. The device runtime instance data is uploaded to the project.

Create a handover configuration report

  1. Read back data is performed to get the current port status.
  1. Go to Building.
  2. Open the Building structure task.
  3. Right-click a hierarchy level and select Reports.
  4. The Select report dialog box opens.
  5. Select the All devices from the project option.
  6. Select IP port used report.
  7. Click OK.
  1. The IP port used report displays.
  2. Devices without Ports and Status information must be upload again.
