Function code and data type definition

Shows the ABT Site input dialog box of an analog input data point with the corresponding analog input selection of function code and data type.

Function code definition

1 = Read coils

2 = Read discrete inputs

3 = Read holding inputs

4 = Read input register

5 = Write single coil

6 = Write single register

15 = Write multiple coils

16 = Write multiple register

 

Data type definition available by the function code

To get more information about the encoding of the respective data point, select the link for the desired data point type.
