Extraction arm, no flow sensor (ExtrcEqp11)

This application function monitors a binary input object (ExtrcSpSel) that activates an extraction arm.

The first configured setpoint value (ExtrcSp1) is selected when the BI is active. The second configured setpoint value (ExtrcSp2) is selected when the BI is inactive.

Required elements and how to configure them:

Element

I/O

Signal type

ExtrcSpSel "Extraction setpoint selector"

On-board output,

Extraction setpoint selector

Binary input

Function

The AF functions as described above. The main output signal is the calculated value from object ExtrcPrSp (Extraction present setpoint).

BI Valid
The validity pin is always True. The application never sets airflow object Invalid.

Main features:

  • Binary input monitor for extraction arm activation
  • Setpoints for extract airflow, configurable
  • Trend for extraction present setpoint
