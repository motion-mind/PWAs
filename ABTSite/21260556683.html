[EvtAckCoord21] Event and acknowledge coordination

Program block, name / node subtype: [EvtAckCoord21] Event and acknowledge coordination

Library hierarchy: Universal functions

Family: EvtAckCoord

Diagram

Configurator

Program blocks are stored in the library without I/Os. Use the auto-create and auto-connect functions to create I/Os and/or to connect them to the interface blocks, e.g., R_A, CMD_B. Alternatively, take the I/Os from the folder containing the predefined I/Os in the library and/or from the plant and connect them to the interface blocks.

Function

This program blocks represents the standard for devices with version < 1.5. For devices with version >= 1.5, use EvtAckCoord22.

Provides up to 8 interfaces for receiving double integer signals about the aggregated alarms, faults, manual interventions and their states from different sources.

It interprets the up to 8 double integer signals and combines them into a new double integer output signal that can be processed with further logic, e.g., event lamp control.

This program block has the following optional functions:

Option: Event interface 1...Event interface 8

Interface to receive information about the aggregated alarms, faults, manual interventions and their states in the form of a double integer value object from another structure.

Option: Acknowledge interface input

Interface to receive a signal for acknowledgement.
