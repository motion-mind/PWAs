2. Configuring BACnet/SC within ABT Site

 

1. Create a hub

  1. The project is available in ABT Site project.
  2. The used Desigo devices need to be at least device version 1.4.
    Note: Device version 1.3 and lower does not support the building hub and the floor hub. However, a flat BACnet/SC structure can always be created with all device versions, down to version 1.2.
  3. Recommendation: The IP network is commissioned and works properly.
  1. Go to Building.
  2. Open the BACnet/SC task.
    BACnet/SC
  3. If you do not see any devices in the Building structure on the left, open Application & device overview and Check for updates.
    Updating device versions
  4. In the Building structure tab, select the device you want to assign as a hub.
  5. Click Assign hub.
  6. A hub is created.
    Note: PXC4 does not support a hub functionality.
  7. More hubs can be created based on the project requirement.
  1. A hub is defined and ABT Site can be setup as BACnet/SC as well.

 

2. Create a failover hub

  1. One or more hubs are available. Creating a failover hub is optional, but it is recommended to define one.
  1. In the Building structure tab, select the device you want to assign as a failover hub.
  2. Right-click and select Properties.
  3. In the Details pane, select BACnet/SC.
  4. In the BACnet/SC drop-down list, select Failover hub.
  5. The device is in an invalid configuration state .
  6. In the Primary hub drop-down list, select the primary hub.
  7. The device is now in the valid failover hub state .

 

3. Create nodes

Create a single node

  1. All required hubs and failover hubs are created.
  1. In the Building structure tab, select the device you want to assign as a node.
  2. Right-click and select Properties.
  3. In the Details pane, select BACnet/SC.
  4. In the BACnet/SC drop-down-list, select Node.
  5. The device is in an invalid configuration state .
  6. In the Primary hub drop-down list, select the primary hub.
  7. It is the valid node with the primary hub state .
  8. In the Failover hub drop-down list, select the failover hub.
  9. It is the valid node with the primary hub and failover hub states .
  10. More nodes can be created based on the project requirement.

Create multiple nodes

  1. In the Building structure tab, select all the devices you want to assign to a hub.
  2. Drag the selected nodes to a hub in the BACnet/SC topology tab.

    Info: You can use multi-selection. Make sure that you place your mouse pointer on the last selected node when you drag the nodes.
  1. The nodes are configured and assigned to the hub and failover hub.

 

4. Create BACnet/SC topology with building- and floor hubs

  1. Hubs and nodes are defined.
  2. The BACnet/SC topology tab shows a flat hierarchy.
  1. In the BACnet/SC topology tab, select a hub, for example, AS_03.
  2. Drag the AS_03 hub over the AS_01 hub in the BACnet/SC topology tab.
  3. The AS_01 hub becomes a building hub.
  4. The AS_03 hub becomes a floor hub.
  5. In the BACnet/SC topology tab, select the AS_08 hub.
  6. Drag the AS_08 hub into the BACnet/SC topology tab below the AS_01 building hub.
  7. This hub becomes a floor hub with the assigned nodes.
  8. Repeat the assignment for all hubs.
  1. All hubs are assigned to the building hub.

 

5. Modify the BACnet/SC network number for each hub network

  1. All devices are assigned to their respective floor hub.
  1. Select a floor hub.
  2. In the SC network column, select the corresponding cell.
  3. Add a new unique BACnet/SC network number.
  4. Click Enter.
  5. All assigned nodes below this floor hub get the same BACnet/SC network number.
  6. Repeat the steps for all floor hubs.
