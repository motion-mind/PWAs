Preparing Modbus meters for Desigo CC graphics

For the graphical representation of a meter in Desigo CC, the data points for the Logical View must be correctly assigned to the meter.

SENTRON PAC2200

Simple meters

  1. A plant, for example, Hydraulic circuit, is available in ABT Site.
  1. In the Plants navigation view, select Hydraulic circuit > Pump.
  2. Right-click and select Add folder.
  3. In the Add folder dialog box, enter a name and a description.
  4. Select the Library view on the right.
  5. From the Field devices > Modbus folder, drag a meter device from the library to the Modbus network.
  6. In the Plants navigation view, select all the data points below the Siemens SENTRON PAC meter.
  7. Drag all the data points to the Electrical meter folder.
  8. In the Plants navigation view, select the Electrical meter folder.
  9. Select the Properties tab on the right.
  10. The Properties view shows the folder properties.
  11. Select the Node subtype property.
  12. Enter the library short name, for example, PAC2200.
    Note: The library extension _MO must not be added.

    Note: The picture does not show all assigned SENTRON PAC data points.
  13. The meter is configured for the Logical View in Desigo CC.
