1. Setting up the devices for IP communication

The IP communication must always be set up first before a changeover to BACnet/SC can take place.

These steps are always necessary, even if a device is added later.

 

1. Setting up the IP communication

  1. Go to Building.
  2. Open the Building structure task.
  3. In the building structure table, select a device.
  4. Open the Details tab.
  5. Configure the detail information of the device.

 

2. Establish network connection

  1. In the Common toolbar, select the arrow next to the Connect icon.
  2. Select the Ethernet tab.
  3. Select the Target selection option IP.
  4. Click Connect.
  1. The IP communication is established and operational.

 

3. Find matching device type

  1. A network connection is established .
  1. Go to Startup.
  2. Open the Configure and download task.
  3. In the Engineered devices area, select the device.
  4. Right-click the device and select Find matching device type.
  5. The devices are listed in the Discovered devices tab.
  6. Select the device and click Assign.
  1. The device is assigned with the device type.

 

4. Download data to the device

  1. A network connection is established .
  1. Go to Startup.
  2. Open the Configure and download task.
  3. In the Engineered devices area, select the device.
    Note: Multi-selection in is allowed.
  4. Right-click the device and select Load > Full (device may restart).
  1. The current device data are downloaded.

 

5. Find matching serial number

  1. A network connection is established .
  1. Go to Startup.
  2. Open the Configure and download task.
  3. In the Engineered devices area, select the device.
  4. Right-click the device and select Find matching serial number.
  5. The devices are listed in the Discovered devices tab.
