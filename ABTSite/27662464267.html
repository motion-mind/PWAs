Replacing charts in programming editor

Replacing charts in programming editor

You can replace charts from the library to the CFC in the Programming editor.

  1. Use an existing program block in the Programming editor.
  1. Go to Library > Program blocks.
  2. Drag a similar program block from the library to the existing program block in the Programming editor.
  3. The Replace program block dialog box displays.
    NOTICE! The new program replaces the old one directly only if both program blocks are compatible, else an incompatibility message displays requesting the user to confirm the replace action.
  4. Click Ok.
    NOTICE! It takes a few seconds for the changes to reflect in the programming editor.
  1. The new program block replaces the old one.
