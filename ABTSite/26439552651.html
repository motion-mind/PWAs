Moving a calendar manager to another device

 

Move a calendar manager to another device

  • 1. In ABT Site:
  1. The current configuration of the project settings must be uploaded first to avoid losing recent changes.
    Uploading runtime instance data
  1. Go to Building > Building structure.
  2. Select the device A where the calendar manager should be stored.
  3. Select the Calendar manager tab.
  4. Click Copy table.

 

  • 2. In Excel:
  1. Launch Excel.
  2. Open an empty sheet.
  3. Right-click and select Paste special.
  4. The Paste special dialog opens.
  5. Select the option Unicode text.
  6. Click OK.
  7. The calendar manager entries are inserted into the Excel table.
  8. Mark the entries and click <Ctrl> + <C> to save the data to the clipboard.
  9. Save the Excel file for later use.

 

  • 3. In ABT Site

Add the calendar manager entries to device B.

  1. Go to Building > Building structure.
  2. Select the new device B where the calendar manager should be stored.
  3. Select the Calendar manager tab.
  4. Activate the Calendar manager check box.
  5. Click Paste table.
  1. The calendar manager table is inserted into the new device.
    Note: At this time, there are 2 calendar managers are in the Desigo system defined but only one calendar manager is allowed.

 

  • 4. In ABT Site

Delete the calendar manager in device A.

  1. Select the previous device A with the calendar manager.
  2. Select the Calendar manager tab.
  3. Clear the Calendar manager check box.
  1. The calendar entries and the global calendar object in device A are deleted.

 

  • 5. In ABT Site

Update device A and device B on site.

  1. ABT Site is online.
  1. Select device A and download the new control program to device A.
    Downloading a control program to a device
  2. Select device B and download the new control program to device B.
    Downloading a control program to a device
  1. The calendar manager is moved from device A to device B.
