Creating a trend log object - EDE import tab

Create a trend log object using the imported EDE data points.

  1. Go to Engineering > BACnet references.
  2. Click the EDE import tab.
  3. Import the EDE files. See Create a BACnet reference from an EDE file import, for more information on how to import an EDE file.
  4. Right-click the required data point in the BACnet references editor area and click Add trendlog.
  2. The trend log object is created in the Plants navigation view.

The device name, device instance number, object type and object instance number information can be seen in the Log device object properties property in the Properties tab.
