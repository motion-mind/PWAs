Replacing I/O data point of the same object type

If another sensor type is available (installed) on the system, the properties can be adjusted by simply replacing it with the new sensor type.

  • Support object types are AI, AO, BI, BO, MO, MI, CI.
  • Not supported are Modbus and KNX PL-Link hardware data points.
  1. Go to Building > Building structure.
  2. Right-click a device and select Go to engineering.
  3. Open the I/O configuration task.
  4. In the I/O configuration area, select a data point in the Tabular view or Graphical view.
  5. Select the Library tab.
  6. Select Field devices > TX & on-board I/Os > [sensor type].
  7. Drag the selected sensor over the existing sensor in the I/O configuration area.
    Info: The tool checks the compatibility of the used sensor.
  1. The sensor is replaced with the new unit range.
  2. Not affected of the replacement are existing settings of the data like:
    - Name, descriptions, alarms, limits, trending.
    - Object ID, and therefore no change to the engineering in the Programming editor is required.
