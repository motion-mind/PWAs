EnableHumRelSetp

EnableHumRelSetp defines if any relative humidity values are displayed.

: Neither the room relative humidity nor the outside relative humidity are displayed.

: The room and/or outside relative humidity (depending on the setting for Enable hum rel display) is displayed. This setting is only meaningful if the sensor for the relative room humidity and/or the sensor for outside relative humidity is enabled.

Disabled: Neither the room relative humidity nor the outside relative humidity are displayed.

Enable Cmf prolong

Enable Cmf prolong defines if the comfort prolongation function for room occupancy is enabled.

: Comfort prolongation is disabled.

: Comfort prolongation is enabled. The user can confirm his presence for a certain time, e.g. 1 h. The time can be configured.
QMX3 interface: TmpCmfIn, TmpCmfRu

Comfort prolongation is disabled.

Enable TP setup shift

Enable TP setup shift defines if the temperature setpoint or the shift of the temperature setpoint is displayed and if the user can change either.

: The temperature setpoint or the shift of the temperature setpoint is displayed and the user can change it. This setting is only meaningful if the room temperature sensor is enabled.

: Neither the temperature setpoint nor the shift for the temperature setpoint is displayed and the user can change neither.

: The temperature setpoint or the shift of the temperature setpoint is displayed and the user can change it. This setting is only meaningful if the room temperature sensor is enabled.
