Favorites

The Favorites task allows you to add a custom favorites list of data points and properties during application configuration. The custom favorites list enables you to view the most frequently used parameter values, and test and share the list of the data points and parameters with other users.
Creating a list of favorites

You can also generate a list of custom favorites automatically.
Generating a list of favorites automatically

You can export and import custom favorites lists
Exporting and importing favorites

You can load the custom favorites list to the controller and view the data points and properties online.
Favorites (online)

You can change parameters of devices with preconfigured applications online.
Changing favorite parameters online

You can change parameter values online per automation station in Startup > Favorites (online). But the parameter list can only be changed in Configuration > Favorites.

② Favorites tree

Displays the list of custom favorites.

Command

Description

Add favorite

Opens the Add favorites window. Enter the name of the favorites list. The command is visible in Edit mode.

Import

Opens a window where you can select a custom favorites list (.viewdescs file) to import. The command is visible in Edit mode.
Exporting and importing favorites

Export

Exports the selected favorites (.viewdescs file) to the default directory for exports.

You can define the default export path in Project Manager > Settings > Export path.
Settings

Context menu

Context menu: Select the favorite item from the Favorites tree and right click to perform the following:
Delete: Deletes the selected favorite item in the Favorites tree.
Duplicate: Creates a duplicate favorite item in the Favorites tree.
Rename: Allows you to edit the name of the favorite item in the Favorites tree.
