Assign/re-assign a BA object to an xFB

A compatible BA object can be reassigned using drag-and-drop in the room control program. In case of existing connections, it will be overridden with the new BA object.

Assign a BA object to an xFB

  1. xFB is not connected to a BA object.
  1. In the Project tree, select the control program.
  2. The BA objects list and the control program are visible.
  3. In the Add control program list, select the BA object.
  4. Drag the BA object over the xFB.
  1. The connection between BA object and xFB is established.

 

Re-assign a BA object

  1. A BA object is assigned to an xFB.
  1. In the Project tree, select the control program.
  2. The BA objects list and the control program are visible.
  3. In the Add control program list, select the BA object.
  4. Drag the BA object over the xFB.
  5. A Delete BA objects dialog box opens.
  6. Click Yes to replace the existing assignment.
  1. The connection between BA object and xFB is new established.
