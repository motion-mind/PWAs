[CMP_I] Comparator for INT values

Description

This instruction compares two input values and sets the output values as follows:

  • GT = 1 if IN1 > IN2
  • GE = 1 if IN1 ≥ IN2
  • EQ = 1 if IN1 = IN2
  • LE = 1, if IN1 ≤ IN2
  • LT = 1 if IN1 < IN2
