Working with Trend view

You want to create a Trend view, configure it, and print the contents.
Trend view

Create the Trend view

Use the Trend view to analyze the values of the pins at run time.

  1. Go to Programming and click or on the toolbar. See Programming for more information.
  1. In the Plants navigation view, expand the Diagnostic folder. See Plants navigation view for more information.
  2. Double-click Add object.
  3. The Add diagnostic object dialog box displays.
  4. In the Type field, select Trend view. Enter the name for the watch table in the Name field and click OK.
  1. You have created a Trend view. Two screens appear for the Trend view, a table and a graph.
Configure the Trend view

You want to configure the Trend view.

  1. You have added the Trend view.
  1. Select the pins from the chart for which you want to create a Trend view.
  2. Drag the selected pins from the chart to the Trend view.
  3. The pin information displays in the trend table and a corresponding graph displays in the trend graph.
  1. You can see a runtime trend graph of the pin data with respect to time.
Print the Trend view

You want to print the Trend view contents.

  1. You have created and configured a Trend view.
  1. Go to the trend graph and click Print on the toolbar.
  2. In the Print dialog box that opens, enter the comment for the printout.
  3. Click Print.
  4. Select a format for the print and click OK.
  5. A print preview dialog box opens.
  6. Check the contents of the trend graph in the preview and click .
  7. Select the folder to save the print and click Save.
  1. You have printed the contents of the Trend view.
