Duplicating and programming

Use the Room Programming Editor to engineer a room.

There are two ways to open the Room Programming Editor:

  • Select Building > Application & device overview, right-click a device and select Duplicate application template and go to programming from the context menu or
  • Select Configuration > Duplicating and programming, right-click a device and select Go to programming.
