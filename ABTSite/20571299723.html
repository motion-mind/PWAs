② Project information

Column

Description

Name

The project name is read-only (see Project Manager > Projects).

Description

Enter a text for the description (free text, maximal 1024 characters).

Version

Enter the version number of your project (free text, maximal 255 characters).

SAP equipment number

Enter the SAP number of the company (free text, maximal 255 characters).

Location

The location information is visible in Project Manager and can be used for filtering projects

 

Customer

Enter the name of the customer (free text, maximal 255 characters).

Country

Enter the country where the project is located (free text, maximal 255 characters).

Region

Enter the region where the project is located (free text, maximal 255 characters).

Address

Enter the address the project is located (free text, maximal 255 characters).

City

Enter the city name the project is located (free text, maximal 255 characters).

Postal code

Enter the postal code where the project is located (free text, maximal 255 characters).

Custom field 1

It is recommended to always use the free text field (maximal 255 characters) for the same type of information within an organization.

Custom field 2

It is recommended to always use the free text field (maximal 255 characters)for the same type of information within an organization.

Authors

It is recommended to always use the free text field (maximal 1024 characters)for the same type of information within an organization.

The location information is also available in the location column of Project Manager (see Project Manager > Projects).
