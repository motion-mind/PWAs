Configuring the calendar manager the first time (manually)

 

  1. Click Add.
  2. The Add calendar manager mapping dialog opens.
  3. Click Add row.
  4. Enter the data:
    • In the Primary calendar name field, enter a description text.
    • In the Primary calendar device ID field, enter the device ID of the primary device.
    • In the Primary calendar object ID field, enter the object ID of the primary device.
    • In the Subordinate calendar name field, enter a description text.
    • In the Subordinate calendar device ID field, enter the device ID of the subordinate device.
    • In the Subordinate calendar object ID field, enter the object ID of the subordinate device.
      Note: There is no plausibility check of the data entered either when entering or when saving.
  5. Repeat the steps 3 and 4 for each subordinate calendar.
  6. Click OK.
    Note: Cancel deletes all entries and the global calendar object.
  7. Download the calendar mapping into the device.
    Downloading a control program to a device
  8. Trigger the calendar synchronization to the other devices with the Web interface.
    Note: The calendar data is automatically synchronized once a day (during the night).
    Synchronizing of the global calendar objects
