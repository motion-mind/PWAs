Short address

Short address shows the address used to communicate point-to-point with one device. Read only.

The short address is applied in the installation tool.

Range: 0...63

-1: unconfigured

-1

Group address

Group address defines the DALI group number.

Devices with the same DALI group number are controlled by the same Lighting-Output-Object (LgtAOut).

Range: 0...15

-1: device is not part of a DALI group.

0…15: group addresses

Note: This property is not applicable to DALI-2 devices.

-1

Serial number

Serial number shows the serial number of the device. Read only.

The serial number is assigned during installation and commissioning.

Presence detector

The Presence detector table in the Functions exists for all the DALI-2 instances that has an activated presence detector. Set the following properties for all activated instances.

Property

Description

Default value

Neutral zone timer

Neutral zone timer defines the minimum wait time between each input event that the presence detector can detects.

1500

Report timer

Report timer defines the heartbeat update interval if there input value do not change.

30

Reception timeout

Reception timeout supervises the reception of regular event messages when Presence detector (occupancy sensor) communicates with the device.

0
