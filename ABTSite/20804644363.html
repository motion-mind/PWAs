Working with the device chart

For field devices or interface functions, for example Modbus adaptations, a device chart can be created. This device chart shows and controls the corresponding device or interface depending on the function to be used. Object references between plant chart and device chart can or must be created. When saving a device chart to the custom library, all hardware and program information about the device is stored and can be used again (green frame).
