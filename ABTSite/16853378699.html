[WAND_DW] Double word AND operation

Description

This instruction generates double word logic AND operations at the input values. All input value bits of the same significance are logically linked by AND; the result is written to the corresponding output value bit.

The number of IN input parameters can be changed.

Example (two inputs)

IN1

2# 1 1 1 1_0 0 0 0_1 1 1 1_0 0 0 0_1 1 0 1_0 0 0 0_0 0 0 0_0 0 0 0

IN2

2# 1 1 1 1_0 0 0 0_0 0 1 1_0 0 0 0_0 0 1 1_0 0 0 0_0 0 0 0_0 0 0 0

OUT

2# 1 1 1 1_0 0 0 0_0 0 1 1_0 0 0 0_0 0 0 1_0 0 0 0_0 0 0 0_0 0 0 0
