Duplicating RDG2 devices

Before one or more duplicates are created, it is recommended to complete the following tasks for the first RDG2 device to reduce the engineering effort later:
- Application type defined
- Parameter configured
- Heating/cooling demand activated
- Subordinate devices added
- RDG2 device chart modified

  1. Go to Building > Building structure.
  2. Right-click a device and select Go to engineering.
  3. Open the KNX PL-Link task.
  4. In the KNX PL-Link area, select the device.
    Note: Duplicating the device or room and device in the Plants navigation view is not supported.
  5. Right-click the RDG2 device and select Duplicate.
  1. An additional RDG2 device is added.
  2. The parameters and configuration are set according the selected RDG2 device.
  3. The subordinate devices are copied.
  4. The device chart is copied according to the selected RDG2 device.
