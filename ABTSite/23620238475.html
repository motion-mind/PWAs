Integrating Desigo Control Point IP in the BACnet/SC network

Configuration ABT Site – BACnet/SC

Shows the BACnet/SC configuration in ABT Site. BACnet/IP enabled on controller, Floor node, 1st floor and this renders the objects visible on Desigo Control Point and the device can be integrated.

 

Topology and data communication

Illustrates which devices can communicate data as well how the data is transmitted. The goal in this scenario is to implement visualization on a Desigo Control Point. In this example, Floor node, 1st floor as well as its BACnet objects are visible. BACnet/IP is enable to this end in the device's BACnet/SC configuration. This renders the device and its BACnet objects visible on BACnet/IP.

 

The BACnet/SC connection from the building hub to the floor node illustrates a visual communication path to depict functionality only. Data is always transmitted via the floor hub!
