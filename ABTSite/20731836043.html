Assigning hierarchical descriptions for BACnet objects

When the project is connected to a 3rd party management station, Assigning hierarchical descriptions allows to efficiently set the descriptions of the BACnet objects to a hierarchical structure. This can be done across the entire automation station. INFO: Desigo CC, Desigo Control Point and Web Interface do not require this assigning, since they recognize the system hierarchy.

Note: The hierarchical description is defined in the active system language (see that section) and is not translatable with standard system mechanisms.

  1. The hierarchy structure is set to
    Including building structure, plant & folders

    or
    Include only plant & folders.

    Plant automation tab
  2. The plant is assigned to a device and a building hierarchy.
  1. Go to Building > Building structure.
  2. Right-click the automation station and select Go to engineering.
  3. Open the BACnet objects task.
  4. From the Device functions drop-down list, select Set hierarchical descriptions.
  5. The Device function dialog box opens.
  6. Click OK.
  1. In the Description column, the hierarchical description text is added to each object.

Use Undo to reject the changes.
