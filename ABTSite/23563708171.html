Checking the notification classes template in the project

The notification class template must be checked before the first device is created in the building structure.
Notification classes

  1. Go to Settings.
  2. Open the Device defaults task.
  3. Select the BACnet tab.
  4. In the Notification class template section, select the appreciate notification template file from the drop-down list:
    • AMEV
    • Desigo
    • KBOB
    • Customized
  1. The notification class file is verified and will be used for all new devices.

For existing devices, the notification class template must be updated for the devices if necessary.
