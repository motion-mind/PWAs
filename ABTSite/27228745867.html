M-bus value information block (VIB)

The M-bus manager implementation of the BA-Device will support up to 10 VIFE extension fields, for example, the combination of VIF.VIFE.VIFE.VIFE.VIFE.VIFE.VIFE.VIFE.VIFE.VIFE.VIFE to allow the mapping of all theoretical manufacturer specific data points as long as the used data types and encoding formats are in line with standard M-Bus definitions. Codes for value information field

The first block of the table contains integral values, the second typically averaged values, the third typically instantaneous values and the fourth block contains parameters (E: extension bit).
