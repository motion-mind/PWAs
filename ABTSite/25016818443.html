Onboarding the device on the Cloud

 

Log-in to Building X

  1. A Building X – Desigo Engineering Access Tool license is required.
  1. Select https://assets.bpcloudapps.siemens.com/.
  2. The Devices application log-in page opens.
  3. Select Log in / Sign up.
  4. Enter the log-in data.
  5. (Optional) Select the correct company from User name settings.
  6. Select Sites and enter the data for the site.
    Note: If the site does not yet exist, the cloud administrator may need to create a partition first. This may be handled differently from country to country.
  7. Select Add.
  8. Select Add.
  1. The site is created.

 

Adding activation key

  1. The device activation key from the customer is available.
  2. In Building X select Launchpad  > Devices .
  1. Select Sites .
  2. Select the site you want to add your device.
  3. Select Add.
  4. The device is created in the Cloud.
  5. Enter the Activation key. This must be the same activation key as in the Excel table for this device.
  6. Select Validate.
  7. Enter the Custom name of the device. This must be the same device name as in the Excel table.
  8. Enter the Custom description of the device. This must be the same device description as in the Excel table.
  9. (Optional) Confirm the External identifier of the device.
  10. Select Add.
  1. The device is on-boarded.
