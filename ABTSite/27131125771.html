Deleting the calendar manager

NOTICE

No calendar manager backed up data available for a later restore

All global calendar data must be re-entered.

  1. It is recommended to save the entries in an Excel table for later use. Deleting the calendar manager cannot be undone.
    Save the calendar manager entries in an Excel table
  1. Go to Building > Building structure.
  2. Select the device where the calendar manager is stored.
    Note: It is recommended to use the device where the time manager is stored.
  3. Select the Calendar manager tab.
  4. Clear the Calendar manager check box.
  5. The Delete calendar manager dialog opens.
  6. Click Yes.
  1. The entire calendar entries are deleted and cannot be restored anymore.
