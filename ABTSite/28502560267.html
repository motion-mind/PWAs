Desigo CC graphics mapping for RDG2 devices

In order for Desigo CC to display the correct graphic, the corresponding node subtype must be defined for each RDG2 device. The display is controlled by the four application types:

  • Fan application
  • Radiant ceiling application
  • Radiant floor application
  • Heat pump application

Within an application type, the correct graphic can be displayed in Desigo CC using the selected node subtype in the property area, for example, RDG_FAN_2PP_HC.
