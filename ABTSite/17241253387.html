Overview

The ABT Installer application is a Siemens Web Package Assistant (SiWA), a self-extracting performable application that installs ABT Site, ABT Pro and its libraries on your machine. ABT Site is a non-licensed tool to engineer and commission building automation systems and ABT Pro is a licensed tool to engineer building automation systems. The ABT_Installer.exe file, when performed, eases the processes of installation and configuration of ABT by displaying simple read, choose and click options.

Network security

All security measures must reduce the risk of potential network security loopholes or security vulnerabilities against exploitation in:

  • ABT Site
  • ABT Pro
  • ABT Go
  • Desigo PXC4/5/7
  • PXG3
  • CloudLink

WARNING

System manipulation due to unauthorized access

  1. Comply with the following items regarding security loopholes, vulnerabilities, and necessary security measures.
  2. Some system components do not meet current IT security expectations without implementing additional protection measures.
  3. Verify that your current Windows version is up to date with all the latest Microsoft service packs and updates for the ABT tools.
  4. The PXC4/5/7 devices are not sufficiently protected against state-of-the-art IT attack strategies without suitable security measures and are at risk of exploitation through potential security loopholes or security vulnerabilities.
  5. A comprehensive risk assessment found that an appropriate and sufficient degree of IT security can be achieved on PXC4/5/7 devices using suitable and reasonable security measures.
  6. Follow the instructions in the Desigo Cybersecurity Guidelines A6V14142297 and Desigo PXC4/5/7 Cybersecurity Guidelines A6V13813955.
