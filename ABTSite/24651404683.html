Deleting a recipient list entry

This function allows you to delete a recipient who no longer exists from the recipient list.

WARNING

Alarms are no longer forwarded to a recipient

Changes after a project data import from the current project can lead to alarms no longer being forwarded on site afterwards.

  1. No offline changes should be made in ABT Site after the commissioning.
  2. If changes are made in ABT Site, the alarm forwarding on site must be checked and verified again.
  1. Go to Building > Building structure.
  2. Open the Application & device overview task.
  3. Select Device types > Controllers > [device].
  4. In the Details pane of the device, select Instance parameter.
  5. In the Property column, select the Recipient list cell.
  6. Right-click and select Configure recipient list.
  7. The Configure recipient list dialog opens.
  8. Select a recipient list entry.
  9. Click Delete.
  10. Click OK.
