Creating a list of favorites

In order to change parameter values online you must create a list of custom favorites first.
Changing favorite parameters online

You can also generate a list of custom favorites automatically by using the Generate favorite button in Configuration > Application configuration.
Generating a list of favorites automatically

You can change parameter values online per automation station in Startup > Favorites (online). But the parameter list can only be changed in Configuration > Favorites.

Create a list of custom favorites

  1. You have added a device with a preconfigured application.
  1. In Building > Building structure, right-click the device and select Go to configuration.
  2. Open Configuration > Favorites.
  3. In the toolbar, select Edit.
  4. Click Add favorite and enter a name for your favorite.
  5. Click Add objects.
  6. Select the objects you want to add and click Ok.
  1. The selected object is added to the Details list.
