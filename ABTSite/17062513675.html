Updating device instances

This workflow describes how to update device instances in existing projects after installing a new graphics library or application library. New device instances created will automatically use the latest graphics library or application type resources.

Preconditions

  • You must have an ABT Site project to perform this procedure. If you have engineered your device online, you must create a new ABT Site project. Use the same credentials as you have used for your device.
    Setting up an ABT project
  • The latest graphics library must be installed on your machine.
  • The latest firmware is available on your machine.
