Set primary address for a M-bus device

You can set the primary address of a discovered M-bus device that is not assigned. Setting a device address is also possible by the device itself (e.g. via its display/dipswitches or 3rd party tools).

  1. Scan your M-bus network as described in section M-bus network scan.
  1. Go to Building > Building structure.
  2. Open the M-bus task.
  3. Expand the Discovered devices folder.
  4. Right-click the M-bus device details given below the Discovered devices folder.
  6. To delete the device from the list, click Delete.
    Note: Deleting a device removes it from the discovery device list. Refer to the section M-bus network scan to recover it again.
  7. To change the address of the device, click Set primary address
  8. A Set primary address window displays.
  10. Enter the device address in the New address field and click OK.
  1. After waiting a few seconds, the device address is updated and a Command successful message is displayed in the Log view.
