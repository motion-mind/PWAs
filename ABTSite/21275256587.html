Deleting a temporary BACnet/SC root certificate

If all devices are updated with the new Main signed certificate, the Temporary signing certificate can be deleted. When deleted, a new ABT operational certificate is created at the same time. If an external certification authority has previously been used, this process must be repeated.

NOTICE

Communication between devices no longer works

  1. Malfunctions of individual devices or entire systems occur.
  1. All devices must be updated with the latest certificate.
  1. All devices are updated with the current Main signed certificate.
  1. Go to Settings.
  2. Open the Certificates task.
  3. Select the BACnet/SC certificates tab.
  4. In the Details view, click Delete temporary signing certificate.
  1. The Temporary signing certificate is deleted.
  2. A new ABT operational certificate is created.
