Create a BACnet reference from ABT Site projects

You can create BACnet reference objects from any existing ABT Site project that can be found on this PC. You can also use context menu to create a BACnet reference object. This way the required BACnet referencing function is set in one-step. The object is created on the top level of the favorite plant in the Plants navigation view .

  1. Go to Engineering > BACnet references.
  2. Select the Projects tab.
  3. In the upper part you can see the available BACnet objects of the selected automation station.
  4. You can browse different automation stations in the project.
  5. In the lower part you can see the available BACnet objects of the selected automation station.
  6. Click on the selected project to browse between different ABT Site projects which are located on this PC.
  7. Drag and drop an object from the lower part of the window to create a BACnet reference object in the desired folder to the Plants navigation view on the left.
  8. You have created a BACnet reference object.
  9. Repeat step 5 to add more objects.
