Exception schedules examples

  1. The automation station PXC4.. PXC5.. or PXC7 is configured.
  1. Go to Building > Building structure.
  2. Right-click the controller and select Go to programming.
  3. In the control program, select the scheduler object.
  4. Right-click and select Configure.
  5. The Scheduler overview dialog opens.
  6. Select Edit.
  7. The Scheduler details dialog opens.
  8. In Exceptions mode, click Exceptions display a calendar view of exceptions for the current month.
  9. While in the Exceptions calendar, click Add in the upper right corner.
  10. From the type drop-down list, select Weekday.
  1. The New exception dialog opens.

Example of a one-day exception

  1. The New exception dialog is open.
  1. Do the following:
  2. In the Day drop-down list, select a day.
  3. In the Week drop-down list, for example, select Any of the 7 days prior to the last 7 days of this month.
  4. In the Month drop-down list, select a day.
  5. In the Priority drop-down list, select a priority.
    Note: This is the scheduler priority and not the BACnet priority array property. This scheduler priority controls the exception if more than one exception is defined to the same time.
  6. Select Next.
  7. The Day dialog opens.
  8. Select the exception start time and add a value.
  9. (Optional) Select the exception intermediate time and add a value.
  10. Select the exception end time.
  11. Activate the End of exception check box.
    Note: The control program takes over the value that was valid before the exception was created.
  12. Select Save.
  13. Check the expected result.

 

Example of a multiple day exception

  1. The New exception dialog is open.
  1. Do the following:
  2. In the Day drop-down list, select Any day.
  3. In the Week drop-down list, for example, select Any of the 7 days prior to the last 21 days of this month.
  4. In the Month drop-down list, select a month.
  5. In the Priority drop-down list, select a priority.
    Note: This is the scheduler priority and not the BACnet priority array property. This scheduler priority controls the exception if more than one exception is defined to the same time.
  6. Select Next.
  7. The Day dialog opens.
  8. Select the exception start time and add a value.
  9. (Optional) Select the exception intermediate time and add a value.
  10. Select the exception end time.
  11. Activate the End of exception check box.
    Note: The control program takes over the value that was valid before the exception was created.
  12. Select Save.
  13. Check the expected result.
