Coordination folder

The Coordination folder shows all group manager objects, scheduler objects, and calendar objects.
Assigning group members to group managers

Add a grouping object

  1. In the Project tree, select [Plant name] > Resources > Coordination.
    Coordination folder
  2. Double-click Add object.
  3. The Add Coordination object dialog box opens.
  4. In the Object type field, select Group manager.
  5. Complete the required fields.
  6. Click OK.
  7. A new grouping object is created in the Project tree.
  8. Right-click the new created object and select Properties.
  9. Select the BACnet view option.
  10. Modify the group manager properties as required.
  11. Click Close.

The description and the name are linked, i.e., the description matches the name and vice versa. To unlink the description from the name, click the chain icon to the right of the Description and Name fields. To relink the description and the name, click the chain icon again.

Add a calendar object

  1. In the Project tree, select [Plant name] > Resources > Coordination.
    Coordination folder
  2. Double-click Add object.
  3. The Add Coordination object dialog box opens.
  4. In the Object type field, select Calendar.
  5. Complete the required fields.
  6. Click OK.
  7. A new calendar object is created in the Project tree.
  8. Right-click the newly created object and select Properties.
  9. Select the BACnet view option.
  10. Modify the calendar properties as required.
  11. Click Close.
  12. Right-click the calendar and select Properties.
  13. The Calendar opens.
  14. Configure the calendar.
    Exception schedules

The description and the name are linked, i.e., the description matches the name and vice versa. To unlink the description from the name, click the chain icon to the right of the Description and Name fields. To relink the description and the name, click the chain icon again.

Add a scheduler to the Programming editor

  1. Drag a scheduler (R_ASCHED, R_BSCHED, R_MSCHED) from the Function blocks folder in the Library to the Programming editor.
  2. The scheduler appears in the Coordination folder.
  3. Do one of the following:
    • In the Project tree, select the Scheduler.
    • In the control program, select the scheduler function block.
  4. Right-click the object and select Configure.
  5. Configure the schedule program as required.
    Configuring a schedule
  6. Define one time manager for the entire project.
    a. Go to Building > Building structure.
    b. Select a device.
    c. In the [device name] tab, select Time sync.
    d. Select the Time manager extension checkbox.
  1. The time manger is marked with a icon.
