Updating device versions

This procedure describes how to update devices for primary control to new product versions that offer more functions and features.

Check the Programming editor for block library updates.
Programming

Update devices

  1. A newer version of the device is available.
  1. Open Building > Application & device overview.
  2. Click Check for updates.
  3. The system checks for new versions. If a new version is found, Update displays in the Version column of the list.
  4. Click Update for each item you want to update.
  5. The Update dialog box opens. A message displays details about the update, e.g. the type version numbers. An info might display telling you that a firmware update is required.
  6. Select the new version from the New type version drop-down list.
  7. Click Update.
  8. The device is updated to the selected version.
  9. The notification classes settings must be updated and checked.
