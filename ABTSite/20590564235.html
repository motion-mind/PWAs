Configuring and downloading to a device

Before downloading the device configuration to your device, you need to configure the IP address.

Configure the IP address of your device
  1. Go to the Room programming editor.
    Duplicating and programming
  2. Open Device configuration.
  3. In Properties > BACnet/IP interface, select Fixed IP.
  4. Enter the IP address of your device.
  5. In the toolbar, click Save project.
Download the configuration to your device
  1. You have configured the IP address.
  1. Select the device.
  2. In the toolbar, click Download to device.
  3. Select the discovered device.
  4. Click Configure.
  5. Click Load.
  6. The Load preview window displays, and the configuration is checked for errors.
  7. Click Load.
