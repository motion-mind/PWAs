[AFil22] Air filter, with filter operating message

Program block, name / node subtype: [AFil22] Air filter, with filter operating message

Library hierarchy: Ventilation and air conditioning equipment

Family: Fil

Diagram

Configurator

Program blocks are stored in the library without I/Os. Use the auto-create and auto-connect functions to create I/Os and/or to connect them to the interface blocks, e.g., R_A, CMD_B. Alternatively, take the I/Os from the folder containing the predefined I/Os in the library and/or from the plant and connect them to the interface blocks.

Function

Monitors an air filter and generates a maintenance indication when the filter has reached its operating hour limit.

This type of air filter monitoring can be used for special types of air filters, e.g., activated carbon filters, where a differential pressure switch or sensor cannot be used.

This program block is almost identical to Mntn21.
