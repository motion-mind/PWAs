[AhuTCtl22] Sensors, setpoints and temperature control for air handling unit with DRA

Program block, name / node subtype: [AhuTCtl22] Sensors, setpoints and temperature control for air handling unit with DRA

Library hierarchy: Ventilation and air conditioning equipment

Family: AhuTCtl

Project tree

Diagram

Configurator

Program blocks are stored in the library without I/Os. Use the auto-create and auto-connect functions to create I/Os and/or to connect them to the interface blocks, e.g., R_A, CMD_B. Alternatively, take the I/Os from the folder containing the predefined I/Os in the library and/or from the plant and connect them to the interface blocks.

Function

Reads and limits the setpoints for supply temperature control from DRA and contains the temperature sensors.

Calculates the control action (heating/cooling) for the energy recovery aggregates.

Changing the alternative for temperature control from TSuCtr21 is not recommended.

A parameter in the program defines the influence on the plant behavior of an alarm or fault of the supply air temperature sensor. It is preset so that the plant does not switch off in the event of an alarm or fault.

This program block has the following optional functions:

Option: Room temperature (1xAI)

Creates a room temperature. The room temperature can be read from other program blocks and used for other functions (e.g., night cooling, room temperature sustained operation).
