Engineering of PX

Standard delivery

The Desigo PX library for collaboration with room solutions (PXC CRSxx) contains a compound for the support of Human-Centric Lighting (HCL), serving as infrastructure to optimize the engineering workflow.

Principle

The compound for the support of HCL (LgtHclLgt11) provides the schedulers for temperature color and brightness adjustment and the associated ramping functionality for smooth transitions.
LgtHclLgt11

By using this compound, the application engineer can more easily program his project specific HCL functionality for the room over a 24-hour cycle (graph schedules preferably provided by the light planner).

The compound commands the information to the application type CenFnct11 via BACnet references.
