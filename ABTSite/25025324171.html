Configuring list of object property references within a scheduler object

Add external references to the scheduler objects which are created in FPE. The List of object property references property is available in the Properties tab in ABT Site editor.

  1. Add a scheduler object in the Field Programming Editor (FPE). See Add a scheduler to the Programming editor.
  1. Go to Building > Building structure.
  2. Right-click the automation station and select Go to engineering.
  3. Open the BACnet objects task.
  4. Select the any scheduler object from the BACnet objects list.
  5. Open the Properties tab.
  6. Select the BACnet view tab.
  7. Click List of object property references property in the Properties tab.
  8. The List of object property references dialog box opens.
  10. Click Add.
  11. Enter the required values in Device instance, Object instance, and Array index fields respectively.
  12. Select Object type and Property from the drop-down list, or, type the description or number in the Object type and Property. See Hybrid control.
  13. The results are displayed accordingly, then click the required value.
    NOTE: These properties are from the respective device/controller used.
    NOTE: Validation of Device instance, Object instance, Object type, and Property is done by the firmware.
    NOTE: The Object type should be same as the scheduler type object, else the object will not be available in the online mode.
  15. To add more object property references, repeat step 8 and 9.
  16. Click Ok.
  17. To delete an existing object property reference, select the property, click Delete, and click Ok.
  1. You have added object property references to scheduler objects.
