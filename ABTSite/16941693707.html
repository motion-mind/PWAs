Favorites (online)

The Favorites (online) task allows you to view custom favorites. You can customize custom favorites in Configuration > Favorites.
Favorites

You can also search for objects in the work area and change parameter values. Changed parameter values update directly to the online device.

Startup > Favorites is hidden by default. To view online favorites right-click the discovered device and select Show online favorites.
