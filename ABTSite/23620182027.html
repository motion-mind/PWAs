Transition building to BACnet/SC communication

Controllers in the green highlighted field communicate via BACnet/SC. Disabling BACnet/IP communication renders the device invisible via BACnet/IP; it can only be addressed via BACnet/SC. The plant hub (highlighted in purple) communicates with the building hub via BACnet/SC and via BACnet/IP to BACnet/IP devices in its subnetwork. Cascadung hub integrates BACnet/IP devices on the BACnet/SC network via the plant hub without rendering a device or BACnet object visible on the closed BACnet/SC network.

 

The BACnet/SC connection from the building hub to the floor node illustrates a visual communication path to depict functionality only. Data is always transmitted via the floor hub!
