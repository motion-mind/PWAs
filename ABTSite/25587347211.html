Configurator

Program blocks are stored in the library without I/Os. Use the auto-create and auto-connect functions to create I/Os and/or to connect them to the interface blocks, e.g., R_A, CMD_B. Alternatively, take the I/Os from the folder containing the predefined I/Os in the library and/or from the plant and connect them to the interface blocks.

Function

SplyChw24 connects one or more chilled water consumers, e.g., room thermostats (e.g., RDG26XxKN), to the cooling plant.

The most important functions in the program block are:

  • Cooling demand collection and evaluation
  • Chilled water temperature setpoint correction based on the chilled water demand value (0...100 %)

Further information

Cooling demand collection and evaluation

SplyChw24 receives an analog water demand signal from consumers and calculates a binary value for the cooling request output pin [CReq] and for a binary calculated value.

Chilled water temperature setpoint correction based on chilled water demand value evaluation of the consumers

SplyChw24 receives chilled water demand value from consumers.

A linear function (configurable) uses this value to calculate a chilled water supply temperature setpoint correction value between SpCorrTFlHi and SpCorrTFlLo:

The calculated setpoint correction value is added to the base setpoint. The result is the chilled water cooling request setpoint value [SpCReq] (chart output pin) and an analog calculated value.

The suggested setpoint value [SpCReq] can be delivered to the cooling plant or other coordinator(s) (SplyChw23).
