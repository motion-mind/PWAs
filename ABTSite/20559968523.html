[Hcr22x] Heating circuit, precontrol

Name: [Hcr22x] Heating circuit, precontrol (all options)

Library hierarchy: Plants

Hcr22x can be used as a simplified heating circuit for radiators (with a linear heating curve, a setpoint reduction defined by a scheduler, and without room sensor).

Plant operating modes

  • Off
  • On

Plant diagram

Main functions

Main components

  • Collect heating requests
  • Heating curve (implemented with function block CHAR_LIN)
  • Heating limit
  • Flow temperature control
  • Pump
  • Valve
  • Safety temperature limiter and overtemperature detector
  • Flow and return temperature sensors
  • Sensor for outside temperature
