Trend log object

Trend log objects are a set of properties for referenced objects which logs the values of these properties periodically. You can create trend log objects in Engineering > BACnet references > EDE Import or Projects tab.
See Creating a trend log object - EDE import tab and Creating a trend log object - Projects tab.

  • A single or multiple trend log objects can be created.
  • Multiple data points can be selected at one time. For each data point a trend log object will be created .
  • The properties of the trend log objects created can be viewed in the Properties view.
