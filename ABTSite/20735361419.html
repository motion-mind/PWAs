Set default descriptions

  1. Go to Building > Building structure.
  2. Right-click the automation station and select Go to engineering.
  3. Open the BACnet objects task.
  4. From the Device functions drop-down list, select Set default descriptions.
  5. The Device function dialog box opens.
  6. Click OK.
  1. In the Description column, the hierarchical description text is removed from each object (if the part name is picked from the text catalog).

Manually changed text in the Name column are not affected by this functionality.
