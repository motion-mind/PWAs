Power return mode

Power return mode defines the behavior of the light when power returns after a power failure or when the application is restarted.

off: The lights are off.

on: The lights are on.

no change (meaningful for bistable relay output): The actuator keeps the switching state of the moment of power failure.

Range: 0...100 [%]

Power failure mode

Power failure mode defines the behavior of the actuator if the power fails.

off: The actuator is switched off.

on: The actuator is switched on.

no change: The actuator keeps the actual value.

Switch-on mode

Switch-on mode defines the dimming value of the actuator when it is switched on.

last actual value (dimming value in the last ON-State): The dimming value is set to the value before the light was switched off.

value according additional parameter SwitchOnSetvalue: The dimming value is set to the value defined in the property Switch-on value.

last received absolute setvalue: The dimming value is set to the last absolute set-value the actuator received.

Relative On enable

: The light can be switched on using dimming.

: The light cannot be switched on using dimming.

: The light can be switched on using dimming.

Relative Off enable

: The light can be switched off using dimming.

: The light can only be dimmed down to the minimum dimming level using dimming.

Load adaption

Load adaptation defines how the actuator determines the load type of the connected device. The load type determines how the actuator controls the connected ballast.

Note
Which load types an actuator supports is product specific.

automatic: The actuator detects the load type of the ballast automatically. The actuator can detect the load types inductive load, capacitive load and resistive load.

leading edge (inductive load): The load type is manually set to inductive load.

trailing edge (capacitive load): The load type is manually set to capacitive load.

automatic
