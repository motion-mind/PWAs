Creating plants

Start with a plant from the library to get a base I/O configuration. Add additional data points in a second step to the plant.

Use the Programming editor to engineer a plant. To open the Programming editor, go to Building > Building structure, right-click a device and select Go to programming.

Add a plant from the library to the project tree

  1. Select the Library tab.
  2. Select a plant.
  3. Drag the selected plant from the Plants folder into the Project tree.
  4. The plant and all its components appear in the Project tree.
  5. (Optional) Right-click the plant in the Project tree, then select Show in program view to open it in the Programming editor.
