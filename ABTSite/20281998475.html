[AhuTCtl21] Sensors, setpoints and temperature control for air handling unit

Program block, name / node subtype: [AhuTCtl21] Sensors, setpoints and temperature control for air handling unit

Library hierarchy: Ventilation and air conditioning equipment

Family: AhuTCtl

Project tree

Diagram

Configurator

Program blocks are stored in the library without I/Os. Use the auto-create and auto-connect functions to create I/Os and/or to connect them to the interface blocks, e.g., R_A, CMD_B. Alternatively, take the I/Os from the folder containing the predefined I/Os in the library and/or from the plant and connect them to the interface blocks.

Function

Performs temperature control (and/or limits the setpoints for supply temperature control).

Contains the setpoints for heating and cooling and the temperature sensors.

Calculates the control action (heating/cooling) for the energy recovery aggregates.

Cascade control for the extract air temperature (TCasCtr21) or supply temperature control (TSuCtr21) is supported and can be configured (alternatively).

The heating and cooling setpoints are preset for cascade control of extract/room air temperature. For supply temperature control, set the heating and cooling setpoints closer together with a narrow zero-energy band.

A parameter in the program defines the effect on the plant behavior of an alarm or fault of the supply air temperature sensor. It is preset so that the plant does not switch off in the event of an alarm or fault.

This program block has the following optional functions:

Option: Room temperature (1xAI)

Creates a room temperature. The room temperature can be used for cascade control (needs small change in the program) and can be read from other program blocks and used for other functions (e.g., night cooling, room temperature sustained operation).

Option: Seasonal temperature compensation

Shifts the setpoints for heating and cooling for very low and very high outside temperature. It is preset for cascade control. In case of supply temperature control, it must be adapted (there is a suggestion in the program).

A binary calculated value object notifies that a seasonal compensation is currently active.

Option: Setpoint correction

Shifts the setpoints for heating and cooling. This is an analog simple process value which is being read and written to from the device chart of a room unit.
