Trend view

A Trend view allows you to visualize multiple online values in a trend chart. Trends are the recurring samples of data in a graphical format. They show the value of the metric across time. You can see the increase and decrease in trend lines across the graph.

Each data recorded for every value is taken at regular time intervals. The trends are packet information of the digital and analog values of a pin. The graph represents the digital axis data on the Y-axis and Time range on the X-axis. You can use a range of colors to highlight the line depicted on the graph.

Display mode

Displays different graph types.

  • Static (static time range display) - The time interval is not refreshed. You can set any range. This mode can be used to visualize all logged values in detail.
  • Strip (dynamic display) - The time axis always displays the current time on the far right. The duration of the time interval can be changed.
  • Scope (jumping time range display) - The time axis displays a fixed time interval. Once the trend values reach the right edge, the display changes so that the current time is shown on the left edge.
  • Sweep (sweep mode display) - Similar to "Scope". However, the values of the "last sweep" continue to be displayed until they are overwritten by the current values.

Select trend

Select which trend values to be displayed.

The trend manager visualizes the trend to axis assignments. You can hide trends by disabling the corresponding check box. Hidden trends are no longer recorded in the trend chart, but still displayed in the legend in grayed out mode. Select trends to be edited in the trend manager. In addition to a color code, the trends in the trend chart are also provided with a marker to enable their quick identification in the legend and trend manager.

Link axes

When a trend is imported, the new x-axis (for example, axis "B") can be linked to the existing x-axis ( for example, axis "A"). In linked mode, axis "B" moves simultaneously with axis "A", but can be adjusted without interfering with axis "A". This mode allows you to compare historic trends with the current trend.

Print

Prints the currently displayed trends of the trend chart.

/

Lock/unlock

Locks/unlocks the trend range along y-axis. Enable write protection or disable it.

Time

Displays the time for the left edge of the x-axis.

You can add a maximum of 16 Trend views only.
