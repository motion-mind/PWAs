Creating and programming adaptions for Modbus interface

1. Open the programming editor

  1. No Modbus device is available in the Plants navigation view.
  1. Go to Building > Building structure.
  2. Right-click the device and select Go to engineering.
  3. Open the Modbus task.
  4. The Modbus area opens.
  5. Open the Programming editor.
    INFO: This can take a while.
  6. The Programming editor opens, and an empty work area displays.

2. Use Device chart from the library

  1. A device chart in the library is available.
  1. Select the Library tab.
  2. Open the Custom Library folder and select the device.
  3. Drag the device to Plant > Resources > Modbus.
    INFO    : Do not clear the Device chart check box anymore. As soon as the Device chart check box is cleared, the device chart is deleted and the program is lost. When this happens, the Undo function must be used.
  1. The Device is created and ready for modification.
  2. The device and I/O configuration is visible in the Plants navigation view.

3. Define object references to the plant chart

Object references of the plant chart to the device chart (or vice versa) are dependent upon the respective application function.

  1. Select the function block in the plant chart.
  2. Drag a link to the prepared input function block in the device chart.
  3. Make required changes to the device chart as required.

 

Create a device chart and save it to the library

  1. A Modbus network and Modbus device are available.
  2. (Optional) I/O data points are created.
  1. Go to Building > Building structure.
  2. Right-click the device and select Go to engineering.
  3. Open the Modbus task.
  4. Open the Programming editor.
    INFO: This can take a while.
  5. The Programming editor opens, and an empty work area displays.
  6. In the Project tree select the Devices tab.
  7. Select Plant > Resources > Modbus > Modbus device.
  8. All configured BACnet objects are displayed in the Project tree.
  9. Right-click the Modbus device and select the Device chart check box.
  10. An empty device chart opens.
  11. Programming and configuring your device chart in the Plant > Modbus device area.
  12. Save the device chart to the Custom library.
