[HpuItf25] Heat pump interface, enable and setpoint outputs

Program block, name / node subtype: [HpuItf25] Heat pump interface, enable and setpoint outputs

Library hierarchy: Heating equipment

Family: HpuItf

Project tree

Diagram

Configurator

Program blocks are stored in the library without I/Os. Use the auto-create and auto-connect functions to create I/Os and/or to connect them to the interface blocks, e.g., R_A, CMD_B. Alternatively, take the I/Os from the folder containing the predefined I/Os in the library and/or from the plant and connect them to the interface blocks.

Function

Controls an analog output for setpoint and a binary enable output (command) that are sent to the heat pump control unit.

This program block can be used in a program block for heat pumps (for heating only or based on a chiller) to control the outputs that are read by the control unit of the corresponding heat pump.

This program block has the following optional functions:

Option: Defrost state (1xBI)

Reads the signal for defrost state from the heat pump control unit.

Option: Fault (1xBI)

Reads the fault signal from the heat pump control unit and generates an alarm.

Option: Feedback (1xBI)

Reads the feedback signal from the heat pump control unit and indicates if the compressor is running.

Option: Heating/cooling mode (1xBO)

Sends a command for heating or cooling to the heat pump control unit.

Option: Present power (1xAI)

Reads the signal for the present power from the heat pump control unit.
