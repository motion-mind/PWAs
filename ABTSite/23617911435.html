Transition building from BACnet/IP to BACnet/SC

Six PXC controllers and a PX classic controller are installed in a building. The challenge is how to implement BACnet/SC in the building in phases and without longer interruptions to communications. This includes illustrated the type of communications between the individual controllers. We are documenting the progress for the customer.

Project Setup

The existing IP configuration is implemented using a building hub and multiple floor hubs. The controllers are employed as follows and assign hierarchically with ABT Site:

  • One PXC5 controller serves as the buidling hub.
  • < 1st Floor uses a PXC7 controller as the floor hub and 1 – n PXC7 for the nodes.
  • < 2nd floor uses a PXC7 controller as the floor hub and 1 – n PXC4 for the nodes.
  • A PXC5 is installed as the floor hub for the basement. Multiple PX classic controllers are integrated via the basement hub.

All devices are connected via a layer 2 switch and are on the same subnetwork. The network uses a star topology.
