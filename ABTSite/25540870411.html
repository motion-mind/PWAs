Assignment of BA objects to xFB in the Room programming editor

 

  1. A building structure is defined.
    Defining a building structure
  2. There is at least one application template in your project or in your library available for selection. If there is no application template available, click Add application template in Building > Building structure.
    Adding project-specific application templates
  3. Room automation stations are defined.
    Adding devices
  1. Go to Building > Building structure.
  2. Select a room automation station.
  3. Right-click and select Go to configuration.
  4. Open the Duplicating and programming task.
  1. The Room programming editor opens.
