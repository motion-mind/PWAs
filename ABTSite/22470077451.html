Set the BACnet/SC device back to BACnet/IP communication

If a BACnet/SC communication has to be changed back to BACnet/IP communication, the device must be completely purged. Only after a complete purge can the device be reloaded with the BACnet/IP settings.

NOTICE

Warranty expires prematurely

When updating an IP configuration to a BACnet/SC configuration, a clear device1 must be performed which will delete all data on the device. As a result, the warranty of any approved project will expire prematurely. Please contact the operator in advance to find out how the warranty can be retained.

  1. A network connection is established / .
  1. Go to Startup.
  2. Open the Configure and download task.
  3. In the Discovered devices area, select the device.
  4. Right-click the device and select Clear device.
  1. The current device is cleared.
    Note: Clear device resets the device to its factory settings. This command deletes all:
    - Loaded applications (control programs, data points, BACnet objects)
    - All field devices on local field networks (such as TX I/O modules, onboard IO configurations, PL-Link/ DALI/Modbus devices)
    - All network settings
    - All loaded certificates
    - The entire control program
