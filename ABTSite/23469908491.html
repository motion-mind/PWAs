Acknowledge a M-bus alarm

M-bus allows you to acknowledge alarms set for all data points. You can see the object status for the data points, M-bus device and M-bus network.

  1. Before you perform the Acknowledge alarm operation, make sure to configure and load the device. See Configure and download.
  1. Go to Engineering.
  2. Open the M-bus task.
  3. Add a M-bus device using the Add device option and data point using the Add data point option.
  4. Tick/check the alarm for few data points in the workplace table.
  5. Change the High limit values for these data points in the Properties table to a value other than the simulated values.
  6. Click  to go online.
  7. You can see the object status for all data points in the workplace table and the object status of the device in the- M-bus work area.
  8. When the online operation is complete, you can see the alarm signal  for the device and the network in the M-bus work area and the workplace table.
  9. Right-click the M-bus network and click Acknowledge alarm.
  1. You can see the acknowledge alarm status  for the device, network and data point in the M-bus work area and workplace table.
