Web interface operation

The Web interface displays the instances of configured devices for online commissioning and engineering.

  1. The Devices application is open.
  2. The device list opens.
  1. Select Sites, then select the site from the list.
  2. Select the device.
  3. Select the Remote access tab.
  4. In the Web interface pane, select Open.
  5. Connecting is displayed.
  6. Enter the log-in information for the web interface.
  1. The web interface opens. Detailed operating functions in web interface is available in document A6V11938631 Web interface.

A timeout occurs if there is no activity in the session: The password must be re-entered.
