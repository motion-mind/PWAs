Exporting signing certificates for Web

Exports a root certificate (the certificate of the certification authority), in order to install the root certificate to a client, for example, Web browser.

  1. Go to Settings.
  2. Open the Certificates task.
  3. Select the Web certificates tab.
  4. In the toolbar, click Export signing certificates for web.
  5. The root certificate is exported and located, for example, C:\Users\Public\Documents\Siemens\DESIGO\Exports.
