Register Desigo PXC.. devices to Building X

  1. Cloud connections are supported by the devices:
    PXC4/5/7
    PXG3.Wx00-2
  2. You have the activation key from ABT Site. For more information, see: Readback activation key.
  1. In Building X select Launchpad  > Devices .
  2. Select  Sites.
  3. Select the site you want to add your device.
  4. Select Add.
  5. The Add device dialog box opens.
  6. Do the following:
    • Enter the Activation key.
      Note: Each device has its own activation key. The activation key is only available if the device is configured.
    • Select Validate.
    • (Optional) Confirm the Custom name of your device.
    • (Optional) Confirm the Custom description of your device.
    • (Optional) Confirm the External identifier of your device.
    • Select Add.
  7. Repeat from step three for all devices that connect to Building X.
