Event enrollment object

The event enrollment object monitors referenced BACnet properties in other objects. The referenced property can be located in the local device or in another device. For example, event enrollment object can be created for devices which does not support intrinsic alarming.

There are six types of event enrollment (Event-Type) objects:

  • Floating limit
  • Command failure
  • Out of range
  • Unsigned range
  • Change of state
  • Change of bitstring

By default, the Present value is monitored which can be changed later. Each event type has an event parameter associated with it. For example, if the BA object type is Analog Input, then the default event type is Out of range.
