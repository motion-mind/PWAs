Remove BA object from corresponding interface in an application function

Remove a BA object

For an object which is used by the application function, the BA object can be removed.

  1. A BA object is assigned to an xFB.
  1. In the Project tree, select the application function of a control program.
  2. The BA objects list and the control program are visible.
  3. In the Add control program list, select the BA object.
  4. Right-click and select Remove.
  1. The BA object is removed from the application function.

 

Remove an owned BA object

For an owned object which is used by the application function, the owned BA object can be removed.

  1. An owned BA object is assigned to an xFB.
  1. In the Project tree, select the application function of a control program.
  2. The BA objects list and the control program are visible.
  3. In the Add control program list, select the owned BA object.
  4. Right-click and select Remove.
  5. A Delete owned BA objects dialog box opens.
  6. Click Yes to remove the owned BA object.
  1. The owned BA object is removed from the application function.
