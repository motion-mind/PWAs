Set ABT Site as BACnet/SC node

As soon a hub exists, ABT Site can be configured as BACnet/SC node.

  1. In the Common toolbar, select the arrow next to the Connect .
  2. Select the Ethernet tab.
  3. Select the Target selection option IP.
  4. Select the BACnet/SC check box.
  5. In the Hub address drop-down list, select an existing hub.
  6. Click Connect when online.
