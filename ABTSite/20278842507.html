[PrfDet24] Profile determination for 4 generators, 1-stage

Program block, name / node subtype: [PrfDet24] Profile determination for 4 generators, 1-stage

Library hierarchy: Universal equipment

Family: PrfDet

Project tree

Diagram

Configurator

Program blocks are stored in the library without I/Os. Use the auto-create and auto-connect functions to create I/Os and/or to connect them to the interface blocks, e.g., R_A, CMD_B. Alternatively, take the I/Os from the folder containing the predefined I/Os in the library and/or from the plant and connect them to the interface blocks.

Function

Contains a profile table with 4 profiles for 4 aggregates.

An external logic (power control and compensation) generates a signal indicating which row of the profile table is requested. The profile table issues the operating state of the aggregates and sends it to the aggregates. An internal logic defines which of the profiles are active and changes them, e.g., based on the operating hours of the aggregates or based on a manual selection.

The profiles are defined in the function block SELP_8MS (Profile and table value selection for 8 multistate values).

Example of a profile table

Profile 1

Row 1: Aggregate 1=Off, Aggregate 2=Off, Aggregate 3=Off, Aggregate 4=Off

Row 2: Aggregate 1=On, Aggregate 2=Off, Aggregate 3=Off, Aggregate 4=Off

Row 3: Aggregate 1=Fixed, Aggregate 2=On, Aggregate 3=Off, Aggregate 4=Off

Row 3: Aggregate 1=Fixed, Aggregate 2=Fixed, Aggregate 3=On, Aggregate 4=Off

Row 3: Aggregate 1=Fixed, Aggregate 2=Fixed, Aggregate 3=Fixed, Aggregate 4=On

Profile 2

Row 1: Aggregate 1=Off, Aggregate 2=Off, Aggregate 3=Off, Aggregate 4=Off

Row 2: Aggregate 1=Off, Aggregate 2=On, Aggregate 3=Off, Aggregate 4=Off

Row 3: Aggregate 1=Off, Aggregate 2=Fixed, Aggregate 3=On, Aggregate 4=Off

Row 3: Aggregate 1=Off, Aggregate 2=Fixed, Aggregate 3=Fixed, Aggregate 4=On

Row 3: Aggregate 1=On, Aggregate 2=Fixed, Aggregate 3=Fixed, Aggregate 4=Fixed

Profile 3

Row 1: Aggregate 1=Off, Aggregate 2=Off, Aggregate 3=Off, Aggregate 4=Off

Row 2: Aggregate 1=Off, Aggregate 2=Off, Aggregate 3=On, Aggregate 4=Off

Row 3: Aggregate 1=Off, Aggregate 2=Off, Aggregate 3=Fixed, Aggregate 4=On

Row 3: Aggregate 1=On, Aggregate 2=Off, Aggregate 3=Fixed, Aggregate 4=Fixed

Row 3: Aggregate 1=Fixed, Aggregate 2=On, Aggregate 3=Fixed, Aggregate 4=Fixed

Profile 4

Row 1: Aggregate 1=Off, Aggregate 2=Off, Aggregate 3=Off, Aggregate 4=Off

Row 2: Aggregate 1=Off, Aggregate 2=Off, Aggregate 3=Off, Aggregate 4=On

Row 3: Aggregate 1=On, Aggregate 2=Off, Aggregate 3=Off, Aggregate 4=Fixed

Row 3: Aggregate 1=Fixed, Aggregate 2=On, Aggregate 3=Off, Aggregate 4=Fixed

Row 3: Aggregate 1=Fixed, Aggregate 2=Fixed, Aggregate 3=On, Aggregate 4=Fixed

"Fixed" is an operating state like "On" but is interpreted differently in the aggregate, For example, when the operating state is "On", the control unit of the aggregate modulates the power. When the operating state is "Fixed", it stops the modulation and tries to maintain the nominal power.

Row 1 represents the "Off" state of the plant for energy generation. Define the operating state "Off" for all aggregates.

Set the states of the aggregates for the next row so that the total power of the enabled aggregates increases.

The setting "Number of rows" for the profiles in the function block SELP8_MS is important, because it is issued over an output pin and a connection in the program to the external logic for "Power control and compensation". The latter then stops and requests more power (next row) when the last row is reached. Adjust this setting if you adapt the project.

This program block has the following optional functions:

Option: Profile manual

Multistate configuration value for manual selection of the profile.

Option: Profile rotation based on operating hours

A program logic (function block ROT_8) measures the operating hours of both energy generators and selects the profile that starts with the one with fewer operating hours when the function is switched on. The profile change is only allowed for a short period of time, e.g., every Tuesday between 4:00 a.m. and 4:05 p.m.
