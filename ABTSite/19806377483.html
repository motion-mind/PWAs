Generating a list of favorites automatically

You can also generate a list of custom favorites automatically.

The Avail. on AS button in Configuration > Default values is used to make features visible in ABT Site. Now it can also be used to build a list of custom favorites.

 

Generate a list of custom favorites automatically
  1. You have added a device with a preconfigured application.
  1. Go to Building > Building structure.
  2. Right-click the device and select Go to configuration
  3. The Configuration > Application configuration dialog box opens.
    Application configuration
  4. In the Configuration section, select a feature to display the configuration parameters.
  5. Select the parameters that you want to display in the custom favorites list, and select Avail. On AS.
  6. Open Configuration > Default values.
    Default values
  7. Select Avail. on AS for the parameters that you want to display in the custom favorites list.
  8. In the toolbar, click Generate favorite.
    • A new list of custom favorites is generated with the selected parameters. The name of the list is SystemFavorite.
    • In the status bar, a confirmation message displays.
  9. Open Configuration > Favorites in order to see the generated list.
    Favorites
Delete an item from the automatically generated list
  1. You have generated a list of custom favorites automatically
  1. Open Configuration > Favorites.
  2. Select the automatically generated list SystemFavorites.
  3. Click Edit.
  4. Select the property that you want to remove.
  5. Click Remove object, and then click Apply.
  1. The selected parameter is removed from the custom favorites list.
