Importing data

 

 

  1. A BACnet object instance csv.file or a BACnet object names csv.file is available.
    Changing BACnet object instance (CSV)
    Changing BACnet object names and descriptions (CSV)
  1. Go to Building > Building structure.
  2. Right-click an automation station and select Import data.
  3. Select a file to import.
  1. You have imported the csv.file.
