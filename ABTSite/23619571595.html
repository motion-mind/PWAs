Transition Floor node, 2nd floor to BACnet/SC

Configuration ABT Site – BACnet/SC

Shows the BACnet/SC configuration in ABT Site. BACnet/IP is set to Disabled on devices AS_11 – Floor hub 1st floor, AS_12 – Floor node 1st floor and AS_22 – Floor node 2nd floor, while the others are set to Enabled. BACnet/IP routing is disabled (cleared) on all devices.

 

Topology and data communication

Illustrates which devices can communicate data as well how the data is transmitted. Disabling BACnet/IP on device AS_11 – Floor hub 1st floor, AS_12 – Floor node 1st floor, and AS_22 – Floor node 2nd floor renders the devices and BACnet objects invisible from the point of view of BACnet/IP. The devices can no longer be accessed via BACEye and the BACnet references from the XWP PX 100 controller on Floor hub, 1st floor and Floor node, 1st floor, as well as Floor node, 2nd floor, no longer operate (reliability).

 

The BACnet/SC connection from the building hub to the floor node illustrates a visual communication path to depict functionality only. Data is always transmitted via the floor hub!
