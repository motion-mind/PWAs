Adding presence detector subordinate devices

  1. A presence detector manager device is available.
  1. In the KNX PL-Link area, select the presence detector manager device.
  2. Right-click the device and select Add subordinate.
  3. The Add subordinate dialog box opens.
  4. In the Add subordinate dialog box, select the compatible subordinate device.
  5. Change the description and extension as needed, for example, Room 103a.
  6. Click OK.
  7. A KNX PL-Link subordinate device is added.
  8. Repeat these steps according to the project requirements.
    Note: The maximum number of subordinates per manager is limited to 9 devices.
  9. The presence detector subordinate devices are added.
  10. Check the subordinate I/O address in the Properties view > BACnet view, for example, PL-1:DL=1;MS=S;MDL=1; (S for subordinate, MDL for subordinate number)

With the presence detector devices, the parameters of the subordinate devices are not synchronized with the manager device.
