Saving the calendar manager entries in an Excel table

  • In ABT Site:
  1. Go to Building > Building structure.
  2. Select the device where the calendar manager is stored.
  3. Select the Calendar manager tab.
  4. Click Copy table.

 

  • In Excel:
  1. Launch Excel.
  2. Open an empty workbook.
  3. Right-click and select Paste special.
  4. The Paste special dialog opens.
  5. Select the option Unicode text.
  6. Click OK.
  7. The calendar manager entries are inserted into the Excel table.
  8. Mark the entries and click <Ctrl> + <C> to save the data to the clipboard.
  9. Save the Excel file for later use.
