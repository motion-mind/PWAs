Recommended programming pattern applications

The standard device charts available with the PL-Link RDG20..KN & RDG26..KN instances in the Field Device Catalog utilize the permanent instantiated features of the RDG20..KN & RDG26..KN (independent of the RDG20..KN & RDG26..KN configuration), and some typical configurable features (not a maximum solution). This table gives an overview of the recommended programming patterns for the most relevant and configurable features of the RDG20..KN & RDG26..KN. The recommended programming patterns are applied project-specifically to the device chart.
