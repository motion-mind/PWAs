Opening global library

The correct global library usually opens automatically by default. It is also possible to open the global library manually.

Open a global library

  1. In Room programming, go to Global libraries.
  2. Click the Open global library icon.
  3. In Windows Explorer, navigate to:
    Local disk (C:) > Program data > Siemens > Desigo > ABT > Libraries > HQ > RP Libraries
  4. Select the correct library set and confirm with Open.
  1. The selected global library is visible.
