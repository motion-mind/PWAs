Create a BACnet reference from an EDE file import

You can create BACnet reference objects from EDE files from other BACnet devices. You can also use the context menu to create a BACnet reference object. This way the required BACnet referencing function is set in one-step. The object is created on the top level of the favorite plant in the Plants navigation view on the left.

  1. Go to Engineering > BACnet references.
  2. Select the EDE import tab.
  3. Click Import EDE.
  4. This opens a generic explorer browser for the user to select the desired EDE file from the PC.
  5. Select the required EDE file and click Open.
  6. The CSV file information will appear in the EDE import window.
  7. You can delete the imported EDE file by clicking on Delete EDE.
  8. The EDE file is deleted.
  9. The file drop down menu shows the currently opened EDE file, with the naming string: Controller name where the file is saves” > “EDE file name”. The drop-down menu allows the user to select any of the previously used EDE files in the project, stored also in other controllers.
  10. The Creation rules drop down is a menu with multiple selection. It will allow the user to select which parameters can be reused from the EDE file when creating the BACnet reference object.
  11. Drag and drop an object from the EDE import window to create a BACnet reference object in the desired folder to the Plants navigation view on the left.
  12. You have created a BACnet reference object.
  13. Repeat step 8 to add more objects.

Further information

  • Support EDE file import with encoding or separator
