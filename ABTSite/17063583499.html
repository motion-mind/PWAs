Updating engineering data on operating and monitoring devices

This procedure processes and enables updates for offline engineering of operating and monitoring devices.

If this procedure is not performed, the update will be performed on the device which can take a long time.

Update engineering data

  1. Open Building > Building structure.
  2. Right-click the Desigo Control Point device and select Go to engineering.
  3. Engineering > Device assignment opens.
    Device assignment
  4. Click Refresh.
  5. Click Assign devices in order to check if the assignment is still as you engineered it.
  6. Click Engineering in browser.
  7. Check if the required graphics are still enabled. If not, enable the graphics.
  8. Close the browser window.
  9. Close the Engineering tab.
  1. The Java environment of the device is updated.
