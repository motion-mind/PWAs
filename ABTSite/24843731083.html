Configuring event enrollment objects

Configuration of the event enrollment is done in the Properties section. This functionality uses offline and online mode.
See Events.

  1. Go to Engineering > BACnet references and create an event enrollment object, see Creating an event enrollment object - Plants navigation view
  2. In the Properties view, click Event parameters.
    Note:     The Reference device instance number and Reference object type fields are mandatory.

Configuring Command failure event type

See Command failure.

  1. Click on the event enrollment object and go to Properties.
    NOTE: By default the present values, based on the object type are populated in the Properties view.
  2. In Properties, select Command failure in Event type.
  3. Click Event parameters.
  4. The Event parameters dialog box opens.
  6. Set the time in seconds in the Delay field.
  7. In the Feedback reference section, enter the required values in Device ID, Object instance, and Array index fields respectively.
  8. Select the Object type and Property from the drop-down list, or, type the description or number in the Object type and Property. See Hybrid control.
  9. The results are displayed accordingly, then click the required value.
  10. Click Ok.
  1. You have configured the command failure event type.

Configuring Floating limit event type

See Floating limit.

  1. Click on the event enrollment object and go to Properties.
    Note: By default the present values, based on the object type are populated in the Properties view.
  2. In Properties, select Floating limit in Event type.
  3. Click Event parameters.
  4. The Event parameters dialog box opens.
  6. Set the High limit, Low limit values, and the time in seconds in the Delay field.
  7. Enter the value in Neutral zone.
  8. In the Setpoint reference section, enter the required values in Device ID, Object instance, and Array index fields respectively.
  9. Select the Object type and Property from the drop-down list, or, type the description or number in the Object type and Property. See Hybrid control.
  10. The results are displayed accordingly, then click the required value.
  11. Click Ok.
  1. You have configured the floating limit event type.

Configuring Out of range event type

See Out of range and Unsigned range.

  1. Click on the event enrollment object and go to Properties.
    Note: By default the present values, based on the object type are populated in the Properties view.
  2. In Properties, select Out of range in Event type.
  3. Click Event parameters.
  4. The Event parameters dialog box opens.
  6. Enter the time in seconds in the Delay field, and set the High limit and Low limit in the respective fields.
  7. Click Ok.
  1. You have configured the out of range and unsigned range event types.

Configuring Change of state event type

See Change of state.

  1. Click on the event enrollment object and go to Properties.
    Note: By default the present values, based on the object type are populated in the Properties view.
  2. In Properties, select Change of state in Event type.
  3. The Event property states type is enabled.
    Note: Unsigned and state texts are enabled if object remains in the same AS.
  4. Define the property state type from the drop-down list. See the below table.
  5. Click Event parameters.
  6. The Event parameters dialog box opens.
  8. Set the time in seconds in the Time delay field.
  9. Add the alarm values from the drop-down list.
    Note: The drop-down list consists of Inactive and Active alarm values.
  10. Click Ok.
  11. To add more alarm values, click Add alarm values.
    Note: The alarm values limit is 12.
  12. To delete the alarm value, select the alarm value and, click .
  1. You have configured the change of state event type.

Configuring Unsigned event type

Unsigned range is similar to change of state event type. See Out of range and Unsigned range.

  1. Click on the event enrollment object and go to Properties.
    Note: By default the present values, based on the object type are populated in the Properties view.
  2. In Properties, select Unsigned range in Event type.
  3. Click Event parameters.
  4. The Event parameters dialog box opens.
  6. Enter the time in seconds in the Delay field, and set the High limit and Low limit in the respective fields.
  7. Click Ok.
  1. You have configured the out of range and unsigned range event types.

Configuring Change of bitstring event type

See Change of bitstring.

  1. Click on the event enrollment object and go to Properties.
    Note: By default the present values, based on the object type are populated in the Properties view.
  2. In Properties, select Change of bitstring in Event type.
  3. Click Event parameters.
  4. The Event parameters dialog box opens.
  6. Set the time in seconds in the Time delay field.
  7. Select the values from the drop-down list in the Bitmask and Bitstring fields.
    The drop-down list contains the alarm values; In alarm, Fault, Overridden, and Out of service.
  8. Click Ok.
    Note: The alarm is generated when the bitmask and bitstring values are same.
  9. To add more bitstring values, click Add bitstring.
    Note: You can add 5 bistring values.
  10. To delete the bitstring value, select the bitstring and, click .
  1. You have configured the change of bitstring event type.
