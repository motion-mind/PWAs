M-bus tracing

Run a trace of the M-bus network/device and save the M-bus trace data in a defined format.

M-bus tracing is not available through MS/TP devices. It is available for MS/TP devices when connected via the WLAN since we have no bandwidth restriction there.

  1. Go to Engineering.
  2. Open the M-bus task and go online.
  3. Right-click the M-bus device and click Tracing > Start.
  4. A Trace - M-bus network window displays on the right. You can see the data in a raw format. Pause, Stop and Export buttons are available.
  6. Click Pause to make a temporary halt to the tracing operation.
  7. Click Stop to cease the tracing operation. An alternative option is to do a right-click the M-bus device and click Tracing > Stop.
  8. Click Export to save the file. Save the file in .pcapng format and click Save.
  9. The trace data is stored in the .pcapng file.
