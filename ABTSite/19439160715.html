ABT Installer FAQs

How to access ABT Installer?

Go to the start menu of your machine and search for ABT Installer.

What is a good Internet speed to download the ABT Installer setup?

ABT Installer downloads more than 9GB of data from the Internet. A download speed of at least 1Mbps is preferred.

How much time is required to install ABT Site and ABT Pro using the ABT Installer?

ABT Installer downloads around 9.32 GB of files to install the Automation Building Tool. The downloading time varies according to your Internet speed. Installing it on high performance machines requires 40…50 minutes.

What to do if I encounter ‘DOWNLOAD_FAILED’ error?
  1. Check your Internet connection.
  2. Try to install ABT again by using the ABT Installer. If the installation fails again, call customer support.
What to do when I encounter ‘ABT Download server error is not running’?

Click on the following link to learn about the current status of ABT download server.
Industry Online Support.

What to do when the ABT Installer asks to restart my machine multiple times?

The ABT Installers may ask you restart your machine due to one or more of the following reasons:

  1. Your Windows machine was not able to delete the pending restart key.
  2. After you launch ABT Installer, a service or a startup application creates the pending restart key.
  3. A virus scanner obstructs your Windows machine to delete the pending restart key.

Workaround:

To delete the pending restart key complete the following steps:

  1. On your Windows machine, go to regedit > HKEY_LOCAL_MACHINE\SYSTEM > CurrentControlSet > Control > Session Manager.
  2. On the right pane, right click on PendingFileRenameOperations file and click Delete.
What to do when the PG/PC interface settings are empty after installing the Automation Building Tool?

If you encounter that the PG/PC interface settings as empty, complete the following steps:

  1. Go to the location where you have downloaded ABT Installer. The default download location is C:\Downloads\ABT_Installer.
  2. Go to ABT-STEP7PLUSPRO-SEBU > Support and and run the PC_IDENTIFIER_09.02.06.00_01.01.00.09.exe file.
  1. You can now see the PG/PC interface settings.
What to do when my ABT Site uninstallation fails?

On tool uninstallation failure, go to C:\ProgramData\Siemens\Automation\Uninstall and run the Uninstall.exe file. If you are still unable to uninstall your ABT site tool, share the log files from the following locations with customer support.

ABT Tool log files are present at C:\ProgramData\Siemens\Automation\Logfiles\Setup.

Uninstall Log files are present at C:\ProgramData\Siemens\Automation\Logfiles\Uninstaller.

Where is my tool and its libraries downloaded?

The tool and its libraries are downloaded to C:\Downloads.

How do I uninstall the ABT setup?

To uninstall the ABT setup, go to Start menu on your machine, search for ABTInstaller.exe file and run it. You will get an uninstallation option to uninstall the tool along with the libraries.

Patch FAQs

What to do when I encounter ‘Download failed’ error while installing ABT Installer patch?

Try to reinstalling the patch. If the error persists then go to C:\Users\<Username>\AppData\Local\Temp\SSU\ and run the ABT_<Version>_<Build Number>.<Patch Number>.exe file. For example, ABT_5.2_39.0.666.9.exe.

Where, <Version> is the Automation Building Tool version, <Build Number> is the ABT tool build version, <Patch Number>is the ABT Patch version, and <Username> is the username of then machine on which ABT is Installed

How do I verify if a new patch is available for the ABT Installer?
  1. Go to the Start menu on your machine and search for Siemens Software Updates.
  2. Open the Siemens Software Updates and check for new patch.
How do I change the frequency in SSU to check for Patch updates?
  1. Go to the start menu of your machine and search for Siemens Software Updates.
  2. Open the Siemens Software Updates, Click Settings > Edit and update the value of Check for Update field as per your requirement.
