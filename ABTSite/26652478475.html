Adding RDG2 subordinate devices

  1. A RDG2 manager device is available.
  2. A RDG200KN, RDG204KN, RDG260KN, RDG264KN KNX PL-Link device must be configured.
  1. In the KNX PL-Link area, select the RDG2 manager device.
  2. Right-click the device and select Add subordinate.
  3. The Add subordinate dialog box opens.
  4. In the Add subordinate dialog box, select the compatible subordinate device.
  5. Change the description and extension as needed, for example, Room 101a.
  6. Click OK.
  7. A RDG2 subordinate device is added.
  8. Repeat these steps according to the project requirements.
    Note: The maximum number of subordinates per manager is limited to 9 devices.
  9. The RDG2 subordinate devices are added.
  10. (Optional) If the subordinate device controls a different zone than the manager device, the zone parameter must be adjusted. An additional zone demand signal is then created.

    Configuring the RDG2 heating/cooling coil demand with zones
    Adapting the RDG2 device chart for heating/cooling demand
  11. Check the subordinate RDG2 I/O address in the Properties view > BACnet view, for example, PL-1:DL=1;MS=S;MDL=1; (S for subordinate, MDL for subordinate number)

With the RDG2 devices, the parameters of the subordinate devices are synchronized with the manager device.
