Managing BACnet references

You can manage the BACnet references in the Programming editor.

Managing BACnet references in the project tree resource folder

  1. The Programming editor is open.
  1. Open the Resources folder.
  2. Expand BACnet references folder.
  3. A list of BACnet objects opens.
  4. You can drag and drop a BACnet object from the list into the chart of the Programming editor.

Moving BACnet references between different folders

  1. The Programming editor is open.
  1. Right click on the Plants folder.
  2. Click on Add folder.
    • Enter the folder name in the Description field and click Ok.
  3. A new folder will be created.
  4. Open the list of BACnet objects from the Resources folder.
  5. Drag and drop the required BACnet object from the list into the new folder that you created.
  6. You can move BACnet objects between different folders.

Copying BACnet references to custom library

  1. The Programming editor is open.
  1. Click on Library.
  2. Expand the Custom library folder.
  3. Drag and drop the required BACnet object from the Resource folder into the Custom library folder.
  4. You can drag and drop the BACnet object from the Custom library folder to the chart.

Verifying xFB and BACnet references object match

  1. The Programming editor is open.
  1. Expand the Function block folder in the Library.
  2. Drag and drop the required block/object into the chart.
  3. Go to the Resources folder where you have the list of BACnet objects.
  4. Drag and drop the required BACnet object onto the function block that you just created in the chart.
  5. You can see an error message in the Info tab at the bottom of the page.
