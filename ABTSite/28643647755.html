Preparing M-bus meters for Desigo CC graphics

For the graphical representation of a meter in Desigo CC, the data points for the Logical View must be correctly assigned to the meter.

Graphic example SentronPAC for M-bus:

Text example heat meter for M-bus

 

  1. A plant, for example, Hydraulic circuit, is available in ABT Site.
  1. In the Plants navigation view, select Hydraulic circuit.
  2. Right-click and select Add folder.
  3. In the Add folder dialog box, enter a name and a description.
  4. Select the Library view on the right.
  5. From the Field devices > M-bus folder, drag a meter device from the library to the M-bus network.
  6. In the Plants navigation view, select all the data points below the Heat meter.
  7. Drag all the data points to the Meter folder.
  8. In the Plants navigation view, select the Meter folder.
  9. Select the Properties tab on the right.
  10. The Properties view shows the meter properties.
  11. Select the Node subtype property.
  12. Enter the library short name, for example, MtrH.
  13. The meter is configured for the Logical View in Desigo CC.
