Deleting devices

This chapter describes the workflow for hard deleting a device. The device gets completely deleted from the system.

- This procedure cannot be undone.

- You can re-onboard the same device later. This operation requires a factory reset.

  1. You are in  Sites.
  1. Select the site with the device to be deleted.
  2. Select the device.
  3. In the Information tab select Delete.
  4. Confirm your decision.
  1. All data related to device (points, point groups, devices associated with that device, etc.) are permanently deleted from the Cloud
  2. The device no longer pushes any data to the Cloud.
