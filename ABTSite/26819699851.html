Configuring a QMX device

  1. A KNX PL-Link device is available.
  1. In the KNX PL-Link area, expand the KNX PL-Link bus to see the features of the device.
  2. Select the feature to be configured, for example, Temperature.
  3. Select the Configuration tab on the right.
  4. The Configuration view shows the properties of the selected device feature.
  5. Configure the properties in the Configuration views.
  6. The creation of the BACnet objects is driven by the configuration. When changing a property in the Configuration view, the corresponding BACnet object is added in the KNX PL-Link area.
  7. With some device configurations, a shared BACnet object is required for data broadcasting. In that case, an object is created on the KNX PL-Link network and is used by all the devices with the same sensor zone.
    Handling shared objects
  8. Configure all the required properties in the Configuration view.
  9. Once the KNX PL-Link device is configured, it can be downloaded and assigned to match the online device.
    Assigning KNX PL-Link devices (online)
