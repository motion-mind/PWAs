Recommended programming pattern applications

The device charts available with the PL-Link device instances in the Field Device Catalog just correspond to a typical configuration of a PL-Link device (not a maximum solution). For all available and configurable features of the room sensors and room operator units, this table gives an overview of the recommended programming patterns. The recommended programming patterns are applied project-specifically to the device chart.
