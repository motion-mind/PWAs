Recommended engineering pattern applications

The standard device charts available with the PL-Link RDG20..KN & RDG26..KN instances in the Field Device Catalog refer to the features of RDG20..KN & RDG26..KN (BACnet objects owned by RDG20..KN & RDG26..KN) with programming patterns, see chapter "Recommended programming patten applications".
Recommended programming pattern applications

In addition, there are features (BACnet objects owned by RDG20..KN & RDG26..KN) without a related programming pattern, so-called engineering patterns.

Engineering pattern: Configuration object value

  • BACnet object can be configured by an O&M client

Engineering pattern: Monitoring object value

  • BACnet object can be monitored by an O&M client

Engineering pattern: Operation object value

  • BACnet object can be operated by an O&M client
