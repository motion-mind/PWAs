Adding and configuring Modbus data points

  1. The Modbus device is configured.
  1. Go to Building > Building structure.
  2. Right-click the device and select Go to engineering.
  3. Open the Modbus task.
  4. In the Modbus area, select the device.
  5. On the Modbus device, right-click and select Add data point.
  6. The Add data point dialog box opens.
  7. Modify the data point properties.

    Function code and data type definition
  8. Click OK.
  9. Repeat the steps 5 to 7 for all required room operation unit data points.
  10. Create the control program in the device chart area.
