Configuring the calendar manager the first time (with Excel)

 

  • 1a. In Excel (manually):
  1. Launch Excel.
  2. Open an empty sheet.
  3. In ABT Site, use Copy table to get the correct layout for the Excel table.
    Save the calendar manager entries in an Excel table
  4. Paste the ABT Site information into the empty Excel sheet.
  5. Enter all the information for the primary calendar and the subordinate calendar manually.
    Note: Do not use any formatting, such as boldface.
  6. Mark the entries and click <Ctrl> + <C> to save the data to the clipboard.
  7. Save the Excel file for later use.

 

  • 1b. In Excel (with calendar export file):
  1. The calendar export file is available.
    Export the calendar information of the entire project
  2. The Calendar manager check box is activated.
    Activate the calendar manager
  1. Open the exported calendar file with Excel.
  2. Open an additional empty Excel sheet. This sheet is required for mapping the primary calendar and the subordinate calendar.
  3. In ABT Site, use Copy table to get the correct layout for the Excel table.
    Save the calendar manager entries in an Excel table
  4. Paste the ABT Site header information into the empty Excel sheet.
  5. From the ABT Site calendar export file:
    • mark the entries in the first 3 columns (blue frame) and copy into the Excel sheet (Primary calendar area).
    • mark the entries in the first 3 columns (orange frame) and copy into the Excel sheet (Subordinate calendar area).
  6. Mark all the entries with the column header (green frame) and click <Ctrl> + <C> to save the data to the clipboard.
  7. Save the Excel file for later use.

 

  • 2. In ABT Site:
  1. Select the device where the calendar manager should be stored.
  2. Select the Calendar manager tab.
  3. Click Paste table.
  4. The imported information from the Excel table is stored.
  5. Download the calendar mapping into the device.
    Downloading a control program to a device
  6. Trigger the calendar synchronization to the other devices with the Web interface.
    Note: The calendar data is automatically synchronized once a day (during the night).
    Synchronizing of the global calendar objects
