[TDiffStepDn21] Temperature difference for step down

Program block, name / node subtype: [TDiffStepDn21] Temperature difference for step down

Library hierarchy: Universal functions

Family: TDiffStepDn

Configurator

Program blocks are stored in the library without I/Os. Use the auto-create and auto-connect functions to create I/Os and/or to connect them to the interface blocks, e.g., R_A, CMD_B. Alternatively, take the I/Os from the folder containing the predefined I/Os in the library and/or from the plant and connect them to the interface blocks.

Function

Generates a step-down request for the logic for power control and compensation if the main temperature difference (main flow temperature – main return temperature) is too small and goes below a certain setpoint.

Most likely there is a hydraulic balancer (or a storage tank) and the flow temperature is measured at the consumer side and the main return temperature at the generation side.

A small difference between the two temperatures shows that the power of the generation is likely too high and must be reduced.

A parameter in the program defines how the temperature difference is calculated: either between the setpoint for the main flow temperature and the main return temperature, or between the measured main flow temperature and the main return temperature.

A parameterizable delay in the program defines for how long after the plant starts the function is activated, e.g., 15 minutes.
