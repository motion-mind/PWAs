Motor Pulley & Belt Sizing

This guide provides the necessary formulas and a Python code-based calculator to find the driven pulley diameter, required belt length, and resulting belt speed for a motor application.

The Formulas

The calculations rely on a few key formulas to determine the relationships between speed, diameters, and lengths in a two-pulley system.

  • Speed Ratio: The ratio between the motor (drive) pulley and the driven pulley determines the final speed of the driven shaft.
    Ratio = Motor RPM / Desired Driven RPM
  • Driven Pulley Diameter (D2): This formula calculates the required size of the driven pulley to achieve the desired RPM.
    D2 = D1 × Ratio
    Where D1 is the diameter of the motor pulley.
  • Belt Length (L): This formula provides an approximate length for a standard V-belt based on the pulley diameters and the distance between their centers.
    L ≈ 2C + (π(D2 + D1))/2 + (D2 - D1)2 / 4C
    Where C is the center-to-center distance between the two pulleys.
  • Belt Speed (V): This calculates how fast the belt travels in feet per minute (FPM), which is useful for ensuring it's within the belt's operational limits.
    V (FPM) = (Motor RPM × π × D1) / 12

Pulley & Belt Calculator ⚙️

You can use the following Python code to perform the calculations. Simply copy the code into a Python environment and change the values in the "User Inputs" section to match your project.


import math

def pulley_belt_calculator():
    # --- User Inputs (Change these values) ---

    # Speed information in Revolutions Per Minute (RPM)
    motor_rpm = 1750
    desired_driven_rpm = 750

    # Known pulley diameter in inches (typically the motor pulley)
    motor_pulley_diameter = 4.0 # inches

    # Center-to-center distance between the two pulleys in inches
    center_distance = 18.0 # inches


    # --- Calculations ---

    # 1. Calculate the speed ratio
    if desired_driven_rpm > 0:
        speed_ratio = motor_rpm / desired_driven_rpm
    else:
        print("Error: Desired Driven RPM cannot be zero.")
        return

    # 2. Calculate the driven pulley diameter
    driven_pulley_diameter = motor_pulley_diameter * speed_ratio

    # 3. Calculate the belt length
    # L = 2C + 1.57(D2 + D1) + (D2 - D1)^2 / 4C
    term1 = 2 * center_distance
    term2 = (math.pi * (driven_pulley_diameter + motor_pulley_diameter)) / 2
    term3 = (driven_pulley_diameter - motor_pulley_diameter)**2 / (4 * center_distance)
    belt_length = term1 + term2 + term3

    # 4. Calculate the belt speed in feet per minute (FPM)
    belt_speed_fpm = (motor_rpm * math.pi * motor_pulley_diameter) / 12


    # --- Results ---

    print("--- Pulley & Belt Calculation Results ---")
    print(f"Speed Ratio: {speed_ratio:.2f}:1")
    print(f"Required Driven Pulley Diameter: {driven_pulley_diameter:.2f} inches")
    print(f"Calculated Belt Length: {belt_length:.2f} inches")
    print(f"Belt Speed: {belt_speed_fpm:.0f} FPM")
    print("\nNote: Choose standard, commercially available pulley and belt sizes that are closest to these calculated values.")


# Run the calculator
pulley_belt_calculator()

Example Calculation

The calculator would output:

You would then look for a standard driven pulley close to 9.33 inches and a standard belt close to 57.5 inches.

Important Considerations

  • Standard Sizes: Pulleys and belts come in standard sizes. Always select the closest available commercial size to your calculated values. Changing the pulley size will slightly alter your final driven RPM.
  • Belt Tension: The calculated belt length is theoretical. Most systems have a tensioning mechanism (like a sliding motor mount) to achieve the proper belt tension upon installation.
  • System Inefficiency: Expect a small amount of speed loss (typically 1-2%) due to belt slippage. The actual driven RPM may be slightly lower than calculated.
  • Belt Type: The calculations are for standard V-belt systems. Different belt types (e.g., synchronous or timing belts) may have different sizing considerations.
